# Category 6: Format Literacy

## What It Tests

Understanding why specific meme formats carry specific meanings. This tests the ability to explain why a particular format was chosen for a particular message—the "grammar" of meme communication.

## Cultural Competency Required

- Understanding that format constrains and shapes meaning
- Recognition of format-message fit (and misfit)
- Awareness of format conventions and violations
- Understanding that choosing a format is a rhetorical act

## Example Meme Patterns

### Easy: Format-Message Match
**Pattern:** A meme using the correct format for its message type.  
**What model should recognize:** The format amplifies the message. "Distracted Boyfriend" for temptation, "Two Buttons" for dilemmas, etc.

### Medium: Format Subversion
**Pattern:** A meme using a format "wrong" for comedic effect.  
**What model should recognize:** The format violation is intentional. Using a "wholesome" format for dark content creates dissonance that IS the humor.

### Hard: Format Evolution
**Pattern:** A format used in a way that extends its original meaning.  
**What model should recognize:** Formats aren't static. A format can acquire new meanings through creative use. The model should track these evolutions.

### Expert: Meta-Format Commentary
**Pattern:** A meme about meme formats themselves.  
**What model should recognize:** Self-referential content about how formats work. This requires understanding the format well enough to understand commentary about it.

## Difficulty Spectrum

| Level | Characteristic | Example |
|-------|---------------|---------|
| 1 | Basic format ID | Name the format |
| 2 | Format-message fit | Why this format works here |
| 3 | Format subversion | Intentional misuse |
| 4 | Format evolution | New meanings over time |
| 5 | Meta-commentary | Memes about memes |

## Scoring Notes

- Must explain WHY a format was chosen, not just identify it
- Should recognize format constraints (what can/can't be expressed)
- Credit for understanding format "grammar" (where text goes, reading order)
- Must recognize when format choice is conventional vs. creative
- Should understand that formats have "registers" (formal/informal, sincere/ironic)

## Common Format Types

1. **Comparison formats:** Drake, Expanding Brain, Galaxy Brain
2. **Narrative formats:** Multi-panel sequences, Before/After
3. **Reaction formats:** Surprised Pikachu, Blinking Guy
4. **Label formats:** Objects labeled to represent concepts
5. **Text formats:** Nobody:, POV:, Me explaining:
6. **Audio/Video formats:** TikTok sounds, video templates
