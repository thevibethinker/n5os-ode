# Sync Folder

**Purpose:** Bidirectional file transfer between Zo Computer and your personal devices

**Location:** `/home/workspace/Sync/`

## Usage

**Drop files here to:**
- Upload from your devices → Zo Computer
- Download from Zo Computer → your devices

**Syncs with:**
- (Add devices in Syncthing web UI)

---

**Syncthing Web UI:** https://syncthing-va.zocomputer.io
