# Video Ideas

## TikTok: Zo Hotline Demo (Joseph Hotline)

**Concept:** Record a TikTok of using the Joseph Hotline live — showing how Zo can figure out what to do with a tool (Zelle example), then craft the actual prompt for you that you can run on your own Zo to get whatever capability you want.

**Key beats:**
1. Show calling the hotline / using it in real time
2. Zo figures out the task (e.g. "what do I do with Zelle?")
3. Zo generates a ready-to-run prompt
4. Take that prompt → run it on your Zo → capability unlocked

**Why it works:** Demonstrates the meta-capability — Zo isn't just answering questions, it's building you new abilities on demand. Tangible, visual, relatable use case.
