# Email Detection Rules

## Jobs
- Keywords: task, job, follow-up, action
- Route to: file N5/lists/tasks.md

## Attachments
- Types: pdf, docx, xlsx, jpg, png, mp4
- Route to: file N5/knowledge/attachments/{type}/

## Articles/Newsletters
- Patterns: http/https URLs, "newsletter" in subject
- Route to: file N5/knowledge/articles/ and digest

## Other
- Default: file N5/knowledge/inbox/
