# Deep Research

Conduct comprehensive research with proper source validation, citation, and structured synthesis.

## Instructions

**You are conducting deep research. Follow systematic methodology.**

### Phase 1: Define Research Scope (5 min)

**Clarify:**
- Research question (specific, answerable)
- Success criteria (what would constitute a good answer?)
- Time box (how deep to go?)
- Output format (report, summary, decision brief?)

**Example:**
```
Question: What are the best practices for AI persona design?
Success: 3-5 validated patterns with examples
Time: 30 minutes
Output: Structured markdown with citations
```

### Phase 2: Initial Discovery (10 min)

**Search Strategy:**
1. Run 2-3 parallel web searches with different angles
2. Scan top 10-15 results
3. Identify 5-7 high-quality sources
4. Note source types (academic, practitioner, documentation)

**Quality Signals:**
- Depth of coverage
- Recency (when relevant)
- Author expertise
- Peer review or validation
- Real-world examples

### Phase 3: Deep Reading (20 min)

**For each source:**
1. Read thoroughly (don't skim)
2. Extract key claims with evidence
3. Note methodology or reasoning
4. Flag conflicting information
5. Track citations for follow-up

**Take notes:**
```markdown
## Source: {title}
**Author:** {name} | **Date:** {date} | **Type:** {type}

### Key Claims
1. {claim} - Evidence: {evidence}
2. {claim} - Evidence: {evidence}

### Methodology
{how they arrived at conclusions}

### Questions/Conflicts
- {item requiring clarification}
```

### Phase 4: Synthesis (15 min)

**Pattern Extraction:**
- What themes emerge across sources?
- Where do sources agree? (strong signal)
- Where do they conflict? (requires investigation)
- What's missing? (gaps in research)

**Validation:**
- Do claims have evidence?
- Is evidence from credible sources?
- Are there counter-examples?
- What are confidence levels?

### Phase 5: Structure Output (10 min)

**Report Format:**
```markdown
# Research Report: {Topic}

**Date:** {date}
**Research Time:** {duration}
**Sources Reviewed:** {count}

## Executive Summary

{2-3 sentences: key finding + confidence + recommendation}

## Key Findings

### Finding 1: {Pattern Name}

**Evidence:** {N sources support this}
**Description:** {what the pattern is}
**Examples:** {2-3 concrete examples}
**Confidence:** High/Medium/Low because {reason}

[Repeat for 3-5 findings]

## Conflicting Information

{Areas where sources disagree + why + how to resolve}

## Knowledge Gaps

{What we still don't know + where to look next}

## Recommendations

1. {Action item based on research}
2. {Action item based on research}

## Sources

[^1]: {Citation with URL}
[^2]: {Citation with URL}
```

### Phase 6: Validate (5 min)

**Quality Checks:**
- [ ] Research question answered?
- [ ] Key claims have evidence?
- [ ] Sources cited properly?
- [ ] Conflicts acknowledged?
- [ ] Confidence levels stated?
- [ ] Actionable recommendations?

## Anti-Patterns

**❌ Cursory Search**
"Quick Google, summarize top 3 results"

**✓ Deep Search**
Multiple search angles, 10+ sources reviewed, patterns validated

**❌ Speculation Without Evidence**
"This approach is best because..." (no sources)

**✓ Evidence-Based Claims**
"3 of 5 sources report X with examples..." [^1][^2][^3]

**❌ Single Source**
Base entire report on one article

**✓ Triangulation**
Cross-reference 5+ sources, note agreement and conflicts
