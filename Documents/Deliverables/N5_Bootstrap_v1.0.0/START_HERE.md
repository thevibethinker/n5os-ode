## 🛡️ Mobius Maneuver Explained

**Unidirectional support channel** where the mature N5 (this one) guides the new N5 during bootstrap:

- **Parent Zo:** Runs read-only HTTP server at `https://n5-bootstrap-support-va.zocomputer.io`
- **New Zo:** Queries parent for help, configs, troubleshooting
- **Security:** New Zo can only READ, never write/modify parent
- **Purpose:** Mature system teaching itself to be reborn elsewhere

**Available endpoints:**
- `/configs/conditional_rules.md` - Critical Zo settings
- `/help/troubleshooting.md` - Common issues
- `/help/dependencies.txt` - Required packages
- `/fixes/common_issues.md` - Quick fixes
