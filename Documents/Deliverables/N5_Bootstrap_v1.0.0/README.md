## 📖 What is N5?

N5 is a personal operating system that runs on Zo Computer, providing:

- **Knowledge Management:** Capture, organize, and retrieve information systematically
- **Meeting Intelligence:** Transform meeting transcripts into structured insights
- **Slash Commands:** 92 reusable workflows for common tasks
- **Session State:** Maintain context across conversations
- **Safety Systems:** Protect against data loss and errors

**Think of it as:** Your personal productivity OS that lives in Zo.
