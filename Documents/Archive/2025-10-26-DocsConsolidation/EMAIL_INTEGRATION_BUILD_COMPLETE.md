## Complete Email Generation Flow

```
1. Input: recipient email, meeting folder

2. Query Stakeholder Tags
   ↓ (query_stakeholder_tags.py)
   Profile found? → Load verified tags
   Profile not found? → Use defaults

3. Map Tags to Dial Settings
   ↓ (map_tags_to_dials.py)
   relationshipDepth: 0-3
   formality: 1-10
   warmth: 1-10
   ctaRigour: 1-4

4. Map Tags to V-OS Brackets
   ↓ (map_tags_to_vos.py)
   [LD-NET] [A-1] *
   (or empty if N5-only stakeholder)

5. Generate Email Draft
   ↓ (generate_followup_email_draft.py)
   Header: metadata + dial settings + V-OS tags
   Body: [placeholder for v11.0 generator]
   Footer: V-OS tag string

6. Save to Meeting Folder
   ↓
   N5/records/meetings/{folder}/follow_up_email_DRAFT.md
```
