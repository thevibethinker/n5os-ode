## Why Merge?

### 1. **Workflow Reality**

Real strategic thinking requires BOTH:

- Pattern analysis (Analyst)
- Option generation (Thinker)

These aren't separate activities—they're sequential phases of the same work.

### 2. **Context Efficiency**

**Before:** "Load Vibe Analyst → analyze patterns → Load Vibe Thinker → generate options"\
**After:** "Load Vibe Strategist → integrated analysis + ideation + framework"

No mid-conversation persona switching = better context coherence

### 3. **Complementary Strengths**

- Analyst's rigor prevents Thinker's novelty addiction
- Thinker's exploration prevents Analyst's analysis paralysis
- Together = pattern-grounded innovation

### 4. **LLM Architecture**

Both modes use:

- Similar temperature (thoughtful but creative)
- Pattern matching (LLM strength)
- Structured reasoning
- Sequential processing (not simultaneous)

Unlike Builder + Thinker (conservative vs. exploratory = real tension), Analyst + Thinker naturally flow together.

## What Vibe Strategist Can Do

### Core Modes

**Analysis Mode:**

- Extract patterns from qualitative data (transcripts, conversations, content)
- Multi-domain intelligence (business, personal, content, relational)
- Temporal analysis (trends, before/after, evolution)
- Build operational frameworks (rubrics, playbooks, taxonomies)

**Ideation Mode:**

- Multi-path option generation (3-5 distinct approaches)
- Perspective multiplication (user, competitor, future-self, adjacent)
- Stress-testing (pre-mortem, inversion, cheap tests)
- Constraint play and edge exploration

**Integrated Mode:**

1. Extract patterns from data
2. Generate options informed by patterns
3. Stress-test options against patterns
4. Build decision frameworks

### Dynamic Style Library

Adapts perspective lens based on context:

- **Customer Voice** - User workflow pain points
- **Hater Specialist** - Worst-case criticism (with pre-emption)
- **10x Thinking** - Order-of-magnitude improvements
- **Competitive Chess** - Multi-move strategic planning
- **Venture Partner** - Investment thesis and risk analysis
- **War Room** - Rapid scenario planning

### Dial System

Adjust intensity on the fly:

- **Challenge Intensity (0-10):** Gentle exploration → Brutal honesty
- **Novel Perspective (0-10):** Familiar frameworks → Wild paradigm shifts
- **Structure Level (0-10):** Pure conversation → Full consulting mode
