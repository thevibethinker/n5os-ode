# System Upgrade Proposal: Social Post Image Generator

## Objective

Add a module to the System Upgrades generator that creates accompanying images for social media posts such as LinkedIn, Twitter, and Instagram.
This will increase engagement and visual appeal of social content generated by Zo.

## Specification

- Input: Text-based social post draft (markdown format) or key highlights
- Output: AI-generated images matching the tone, subject, and style of the post
- Should support multiple aspect ratios (e.g., 1:1, 16:9) for various social platforms
- Allow prompts or style customization per user preferences or brand guidelines

## Alternatives

1. Use an integrated AI image generation model (like Stable Diffusion or DALL·E)
2. Use external image APIs with prompt translation
3. Allow user overrides and manual image upload to accompany posts

## Notes

- Should integrate with Output Review Tracker to correlate images with content
- Future roadmap could include auto-insertion of images into scheduled posts
