# Conversation-End System - Overview

Version: 2.0
Related: N5/prefs/operations/conversation-end-cleanup-protocol.md

## Purpose

Clean handoffs between sessions, proper artifact placement, workspace hygiene.

## When to Trigger

User says: "end conversation", "close thread", "wrap up"

## Core Protocol

1. Assessment - What was produced?
2. Deliverable Placement - Move finals to proper locations
3. Temporary Cleanup - Process Records/Temporary/
4. Documentation - Create handoff context

## Deliverable Routing

Meeting follow-ups: Personal/Meetings/{id}/DELIVERABLES/
Production scripts: N5/scripts/
Knowledge docs: Knowledge/{category}/
Company records: Records/Company/

## Checklist

Pre-End:
- Identify all deliverables
- Confirm final versions
- Check Temporary/ contents

Execution:
- Move finals to proper locations
- Archive valuable iterations
- Delete scratch files

Handoff:
- Document accomplishments
- Note deliverable locations
- List outstanding work
