# Knowledge/current - Communications Context Folder

## Purpose

This folder contains the **most up-to-date information** about V and Careerspan that should be referenced when generating outbound communications (follow-up emails, blurbs, descriptions, one-pagers).

## What Belongs Here

- **Current positioning documents** - How Careerspan is positioned right now
- **Recent value props** - Latest messaging about what Careerspan does
- **Active initiatives** - What V/Careerspan is working on currently
- **Recent wins/updates** - Latest traction, customers, milestones
- **Messaging guidelines** - Tone, voice, key talking points
- **Templates** - Example emails, blurbs, one-pagers that work well

## How This Is Used

When the **Communications Generator** agent runs (after meeting intelligence blocks are complete), it:

1. Loads the full meeting transcript
2. Loads B14 (blurbs requested) if exists
3. Loads B25 (deliverables map) if exists
4. **Loads everything in this folder** as context
5. Generates send-ready communications

## Usage Pattern

**You (V) should:**
- Drop any document you want the communications generator to reference here
- Update docs here as positioning/messaging evolves
- This is your "one-place dumping ground" for communications context
- Don't worry about organization - just put relevant docs here

**The system will:**
- Always reference this folder when generating emails/blurbs
- Ensure generated communications align with current Careerspan positioning
- Use this context to avoid outdated or misaligned messaging

## Examples of What to Drop Here

- `careerspan_positioning_2025.md` - Current market positioning
- `value_prop_talking_points.md` - Key benefits to emphasize
- `recent_wins.md` - Latest traction to reference
- `email_examples.md` - Emails that worked well
- `blurb_templates.md` - Successful blurb formats
- `messaging_guidelines.md` - Tone, voice, do/don't

## Why This Exists

Previously, the communications generator would need to search across the entire Knowledge folder, which could be overwhelming. This folder provides a **focused, guaranteed context** that's always up-to-date because you maintain it manually.

Think of it as: "If I had to brief someone on Careerspan in 5 minutes before they wrote an email on my behalf, what would I tell them? Put that here."
