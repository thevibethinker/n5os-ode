## Aggregation Methodology

**Process:**
1. Select meetings by stakeholder type (sales customers, community operators, advisors, etc.)
2. Load B31 files for initial insights
3. Cross-reference against full transcripts (1000-char context windows)
4. Extract 1-2 best quotes per insight (3-4 sentences max)
5. Detect cross-meeting patterns (3+ meetings = validated pattern)
6. Generate categories dynamically based on emergent themes
7. Synthesize actionable recommendations

**Quality standards:**
- Quotes must be verbatim from transcript
- Context provided for all quotes
- Signal strength rated per insight
- Cross-meeting patterns explicitly noted
- Recommendations tied to specific evidence
