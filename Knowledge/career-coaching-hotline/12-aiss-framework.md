# The AISS Framework

## Action, Impact, Scale, Skill

AISS is a framework for writing resume bullets that demonstrate value rather than describe duties. Every strong bullet answers four questions.

## The Four Components

1. **Action** — What did you do? Start with a strong past-tense verb. Not "responsible for" or "helped with," but "led," "designed," "reduced," "launched."
2. **Impact** — What was the result? Quantify whenever possible. Revenue generated, costs reduced, time saved, users served, errors eliminated.
3. **Scale** — What was the scope? Team size, budget, user base, geographic reach, number of systems. Scale provides context for impact.
4. **Skill** — What capability does this demonstrate? The skill should map directly to something the target role requires.

## Examples

**Weak:** "Managed social media accounts for the company."

**AISS:** "Grew organic social media engagement by 340% across 4 platforms over 6 months, demonstrating data-driven content strategy and audience development."

- **Action:** Grew organic engagement
- **Impact:** 340% increase
- **Scale:** 4 platforms, 6 months
- **Skill:** Data-driven content strategy, audience development

## Application Tips

- Not every bullet needs all four components, but aim for at least three.
- Impact without scale is unanchored. "Saved $50K" means different things at a startup versus a Fortune 500.
- Skills should map to the job description you are targeting — this is where tailoring meets bullet writing.
- When you lack hard numbers, use directional impact: "significantly reduced," "doubled," "first-ever."
