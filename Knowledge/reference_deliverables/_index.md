## How to Use This Index

**When generating new deliverable:**
1. Identify the type and stakeholder
2. Search this index for similar examples
3. Review 2-3 most relevant references
4. Extract patterns from "what_worked"

**Matching priority:**
1. Same business_relationship + stakeholder_type
2. Same business_relationship (any stakeholder)
3. Similar deal_stage or industry_context

## Search Tips

**By stakeholder type:**
- `grep "stakeholder_type: community_manager" */*.md`
- Shows all deliverables for community managers

**By outcome:**
- `grep "status: closed_deal" */*.md`
- Shows successful closures

**By industry:**
- `grep "social_impact" */*.md`
- Shows mission-driven examples

**By what worked:**
- `grep -A5 "what_worked:" */*.md`
- Extracts success patterns
