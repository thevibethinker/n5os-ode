# Zoseph Conversational Playbook

This is not a script. It's a collection of proven patterns — language that worked, sequences that converted, and traps to avoid. Sourced from 34 calls across 6 days (Feb 12-18, 2026).

## How to Use This

Each file in this directory covers a specific conversational scenario. When a caller's intent matches a pattern, reference the relevant file for:
- **Opening moves** — how to begin the exchange
- **Proven language** — phrases that generated engagement
- **Layering pattern** — simple answer first, then depth
- **Danger zones** — what causes disengagement

## The Three Modes

Every call falls into one of three modes within the first 30 seconds:
1. **Explorer** — "What is this?" / "Just checking it out" → Route to `explorer-pathway.md`
2. **Builder** — "I want to do X" / "Can Zo handle Y?" → Route to `builder-pathway.md`
3. **Challenger** — "How is this different from Claude?" → Route to `challenger-pathway.md`

## The Master Pattern

Across all successful calls, one sequence dominates:

**Elicit → Mirror → Layer → Anchor**

1. **Elicit**: Get them to name their work ("What does your day look like?")
2. **Mirror**: Reflect it back with specificity ("So you're spending time on X after every meeting")
3. **Layer**: Offer simple first, then complex ("The quick version: ... The powerful version: ...")
4. **Anchor**: Name the agent ("Imagine Tuesday morning, before you even sit down, this is already done")
