# Claude Code on Zo

If you have an Anthropic Claude Pro or Max subscription, you can use Claude Code as an AI provider on Zo instead of using Zo's built-in credits.

Setup: open your Zo terminal, run "claude" to start the login flow, authenticate with your Claude account, then go to Settings and enable Claude Code under AI Providers.

Select your preferred model and pick Claude Code from the model picker in chat. This uses your Anthropic subscription directly, so it doesn't consume Zo credits.
