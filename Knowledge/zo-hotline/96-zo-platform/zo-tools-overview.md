# Zo's Tools

Zo has over 80 built-in tools organized into categories:

Files: create, edit, read, search files. Generate and edit images. Generate video. Create diagrams. Transcribe audio and video.

Web: read, open, browse, and save web pages. Search the web, X/Twitter, Google Maps, and find similar links.

Computer: run shell commands, create and manage sites and services, proxy local services, change hardware specs.

Integrations: send email and SMS, use Gmail, Google Calendar, Google Drive, Notion, Airtable, Linear, Dropbox, Spotify, OneDrive, and Telegram.

Commerce: create Stripe products, prices, payment links. Manage orders.

Agents: create, list, edit, delete scheduled tasks.

Personalization: manage personas, rules, and user settings.

Zo Space: create and manage pages and API routes on your personal space.

You can mention specific tools by typing @ in chat.
