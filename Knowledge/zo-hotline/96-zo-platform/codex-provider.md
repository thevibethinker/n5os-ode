# OpenAI Codex on Zo

If you have an OpenAI Plus or Pro subscription, you can use Codex as an AI provider on Zo instead of using Zo's built-in credits.

Setup: open your Zo terminal, run "codex", select Sign in with Device Code, and follow the authentication flow. You may need to enable device code authorization in your ChatGPT settings first.

Then go to Settings, enable Codex under AI Providers, and select it from the model picker. This uses your OpenAI subscription directly.
