Every Zo user gets 100 gigabytes of cloud storage.

Zo works with most file formats. Notes are stored as markdown. Spreadsheets can be CSV, Excel, or Zo's built-in sheet format. Code files support syntax highlighting for most programming languages. Word documents can be converted to notes. Audio and video files can be transcribed. Images can be displayed and generated. PDFs and ebooks can be read and discussed.

If Zo can't edit a file directly in the browser, you can ask it in the chat to edit or convert the file. Because Zo has full access to a Linux computer and can install tools, it handles a surprising range of file types.

You can add files by dragging and dropping in the web app, uploading through the interface, importing from connected services like Google Drive or Notion, or syncing from your desktop using the Zo desktop app.

Zo also saves web articles automatically. Just send a URL in the chat or via text message and Zo saves the page to your workspace.
