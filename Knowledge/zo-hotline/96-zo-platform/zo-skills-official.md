Official Zo Skills

These are built by the Zo team and come ready to use.

Sites & Content:
- Create a portfolio site. Build a personal portfolio to showcase your work and resume.
- Create a site. Create a site hosted on your Zo server.
- Create a guestbook. Build an interactive guestbook where visitors can leave messages.
- Create a blog. Build a blog or newsletter to share your thoughts.
- Create a slide deck. Turn any reference material into a polished, visually cohesive slide deck.

Productivity & Automation:
- Find meetings with attendees. Shows you all the people you've met with recently, organized by meeting.
- Daily news digest. Fetch the latest headlines on a topic you care about and email it to you every morning.
- Daily Linear digest. Creates an automation to email you a summary of your Linear activity from the previous day.
- Enrich CSV with AI-generated columns. Adds new columns to your spreadsheet by generating information for each row.
- Automate something. Create an automation with a Zo Agent task.
- Organize my files. Organizes your scattered files into logical folders by project, type, and purpose.
- Text summary of important emails. Finds your most important emails from the past 2 days and texts you a summary with suggested follow-ups.
- Process file line-by-line with script. Processes every line of a file one by one, transforming or extracting information from each line.
- Weekly meeting reminder. Get an email every Sunday evening listing your meetings for the coming week.

Files & Documents:
- Resize Images. Resize images to specific dimensions or percentages.
- Extract Text from Images (OCR). Extract text from images, scanned documents, and PDFs using OCR.
- Convert EPUB to Markdown. Convert EPUB ebooks to Markdown format.
- Generate PDF from Markdown. Convert Markdown files to PDF format with professional formatting.
- Convert PDF to Markdown. Convert PDF files to Markdown format.
- File Share. Share files with email-gated access.

Developer Tools:
- Research a topic. Use Zo to research any topic comprehensively and write a report.
- Set up Grafana for log visualization. Sets up a visual dashboard to explore and analyze your system logs and data.
- Set up n8n workflow automation. Sets up n8n workflow automation tool so you can create automated workflows between different apps and services.
- Google Direct Oauth. Direct Google OAuth integration without third-party apps.
- Set up SSH server for remote access. Sets up secure remote access to your Zo Computer so you can connect from anywhere.
- Set up code-server (VS Code in browser). Sets up VS Code in your web browser so you can write and edit code from anywhere.
