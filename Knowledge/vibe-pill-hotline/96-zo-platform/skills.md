Skills are packaged workflows that Zo can follow. Think of them as recipe cards for your AI.

A skill is a folder containing a SKILL dot md file with instructions, plus optional scripts, reference docs, and templates. When a task matches a skill's description, Zo reads the instructions and follows them.

Skills are reusable, shareable, and self-documenting. You can create your own skills or install them from the Zo skills registry.

This is Zo's approach to custom integrations too. Instead of a rigid plugin system, Zo builds skills that combine instructions, code, and context to extend what it can do.
