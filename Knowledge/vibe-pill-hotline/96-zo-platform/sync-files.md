# Sync Files to Your Zo

The Zo desktop app has built-in file syncing. Pick a folder on your computer and it stays in sync with your Zo workspace automatically. Changes on either side sync to the other.

Alternative: you can use SyncThing, a free open-source sync tool. Install it on both your computer and your Zo, connect them via device IDs, and create a shared folder.

Important: synced folders on Zo must be inside /home/workspace to be visible in the Zo app. If you use SyncThing, make sure to password-protect it on your Zo since the service URL is publicly accessible.
