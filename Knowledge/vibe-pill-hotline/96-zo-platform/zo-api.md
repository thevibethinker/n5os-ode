# Zo API

Zo has a full API you can use from scripts, automations, or other apps. You get an access token from Settings, then Advanced, and use it to call endpoints.

The main endpoint is POST to api.zo.computer/zo/ask — you send a message, get a response. It's like texting Zo but from code.

You can also stream responses in real-time, manage agents, control the browser, work with personas and rules, and manage your zo.space — all through the API.

This is powerful for developers who want to build on top of Zo, create custom integrations, or automate workflows that go beyond what the built-in tools offer.

The API reference docs cover every endpoint with examples.
