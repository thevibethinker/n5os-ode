# Keanu to Market — GTM Wisdom Gift Site

A parody gift site that filters go-to-market wisdom through Keanu Reeves' chill zen energy. Built on Zo Computer as a fun side project.

## What It Is
A website where you can explore GTM (go-to-market) concepts reframed through Keanu's philosophical style. Part novelty, part genuine startup advice.

## Why It Exists
Demonstrates how Zo Computer can be used to build and deploy full web apps — from concept to live site — entirely through conversation-driven development.

## Built With
Zo Computer Sites system — staging → production workflow, deployed as a hosted service.
