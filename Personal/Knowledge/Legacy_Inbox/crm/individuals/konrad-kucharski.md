## Aviato API Intelligence

**API Overview**: Aviato provides a comprehensive RESTful API for people and company data enrichment

**Authentication**: Bearer token authentication
- API keys managed via team dashboard
- Requires account setup at https://data.aviato.co/

**Person Enrichment Endpoint**: `GET https://data.api.aviato.co/person/enrich`

**Query Parameters (support multiple ID formats)**:
- `linkedinID` (slug from /in/ URL)
- `linkedinEntityId` (numeric ID)
- `linkedinURL` (accepts public, Sales Navigator, Recruiter URLs)
- `email`
- `twitterID`, `crunchbaseID`, `angelListID`, `signalNfxID`, `polyworkID`
- `rescrape` (boolean) - trigger fresh scrape
- `maxDataAgeDays` (integer) - staleness threshold

**Data Returned** (90+ datapoints):
- Full name, headline, gender, location (granular geocoding)
- Work experience (company, positions, dates, departments, seniority scores)
- Education (schools, degrees, field of study, dates)
- Skills, languages (with proficiency levels)
- LinkedIn metrics (connections, followers, join date, labor status: OPEN_TO_WORK / HIRING)
- Investor data (if applicable): interests, disinterests, investment range, portfolio
- All associated URLs (LinkedIn, Twitter, Crunchbase, AngelList, Signal NFX, personal website)
- Computed highlights (proprietary AI-driven insights)

**Company Enrichment Endpoint**: `GET https://data.api.aviato.co/company/enrich`
- Similar structure for companies
- 240+ company datapoints including credit card revenue, web traffic, tech stack, funding, valuation, employee vesting schedules

**Pricing Model**: Sales-driven (requires demo/contact with team)
**Integration**: Available via API, Clay.com integration, direct CRM integrations
