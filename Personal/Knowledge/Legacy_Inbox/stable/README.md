# Knowledge: Stable Layer

This directory contains **historical and biographical knowledge** that changes rarely.

---

## Purpose

Stable knowledge represents foundational truths that remain constant over time:
- Biographical information (who V is, background, expertise)
- Company founding story and mission
- Historical milestones and timeline
- Core values and principles

---

## Files

- **`bio.md`**: Founder profiles, motivations, expertise
- **`company.md`**: Careerspan comprehensive information
- **`timeline.md`**: Chronological milestones and events

---

## Characteristics

- **Changes Rarely**: Updated only when fundamental facts change
- **High Confidence**: Information is verified and authoritative
- **Historical Truth**: Represents what happened, not current state
- **Portable**: Can be exported and remain interpretable

---

## Usage

Referenced by:
- AI persona initialization
- Context loading for conversations
- Company documentation generation
- Pitch deck and marketing materials
