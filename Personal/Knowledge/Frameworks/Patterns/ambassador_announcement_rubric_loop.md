# Pattern: Ambassador Announcement Rubric Loop

**Use when:** Crafting high-leverage announcement posts (new role, launch, milestone) where tone, positioning, and call-to-action all matter.

## Core Idea
Turn "write a great announcement" into a short feedback loop:

1. **Define the job of the post** as a rubric (3–5 criteria, each 1–5).
2. **Translate each rubric line into constraints** (who, what, why now, for whom).
3. **Generate multiple concepts** that explicitly trade off different dimensions (e.g., one more spicy, one more proof-heavy).
4. **Score each concept against the rubric** instead of vibes.
5. **Iterate the strongest concept** until it scores ≥4 on every critical dimension.

## Steps

1. **Rubric first, copy second**
   - Example criteria: Hook strength, POV clarity, Zo positioning, social proof, CTA.
   - Write them *before* you draft.

2. **Concepts, not drafts**
   - For each concept, write: angle, target audience, emotional temperature, sample 1–2 lines.
   - Intentionally vary angle: "philosophy-forward", "proof-of-work", "founder-to-founder".

3. **Score + diagnose**
   - 1–5 per rubric dimension.
   - Add 1–2 bullet notes: "Hook too abstract", "CTA missing for ideal reader".

4. **Merge + sharpen**
   - Take best-scoring concept.
   - Pull best line(s) from others to upgrade weak dimensions.

5. **Final readout**
   - Only ship when: no dimension <4 *and* there is one clearly memorable line you’d be happy to be quoted on.

## Why it works
- Forces explicit tradeoffs instead of "vibes only" drafting.
- Keeps announcement aligned with positioning (e.g., AI-age productivity, non-technical builder) instead of generic "excited to share" energy.
- Gives a repeatable loop you can re-use for future launches and thought-leadership posts.
