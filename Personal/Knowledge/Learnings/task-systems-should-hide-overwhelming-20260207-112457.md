# Task systems should HIDE overwhelming backlogs, not display them. Visual clutter...

Task systems should HIDE overwhelming backlogs, not display them. Visual clutter physically degrades neural efficiency in ADHD brains (Yale 2024). Focus interfaces on 'What's next?' not 'What's pending?'
