# Idea Lab: Structured Exploration System

## Purpose
The Idea Lab is a dedicated environment for taking "sparks" (from `file 'Lists/ideas.jsonl'`) and subjecting them to rigorous cognitive stress-testing. This system is designed for V's "Warrior" (COMT GG) architecture—providing a high-complexity, structured sandbox for deep thought without the risk of unstructured mental leakage.

## The Pipeline

1. **Capture:** Ideas start in `file 'Lists/ideas.jsonl'`.
2. **Promotion:** High-potential ideas are "promoted" to an Exploration folder under `file 'Personal/Knowledge/Lab/Explorations/<slug>/'`.
3. **Stress-Testing:** The ideas are explored through specific semantic modules (Socratic, Adversarial, First Principles).
4. **Crystallization:** The final result is documented in `Synthesis.md` and potentially promoted to `file 'Personal/Knowledge/Canon/'`.

## Directory Structure
- `Explorations/`: Active and archived thought experiments.
- `Templates/`: Canonical structures for different exploration modes.
