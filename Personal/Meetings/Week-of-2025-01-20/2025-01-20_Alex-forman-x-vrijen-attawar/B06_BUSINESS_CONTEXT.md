# Business Context

## Parse (Alex's Company)

**Product:** Turns any website into an API
**Value Prop:** Democratizes access to web data without technical scraping knowledge

**Target ICP:**
- Non-technical users who need data from websites
- Industries with no existing developer ecosystem (e.g., garbage collection)
- People with domain expertise but no technical skills
- Users enabled by AI tools (Claude, etc.) to build software

**Business Model:**
- **Current:** Monthly credit buckets
- **Transitioning To:** Usage-based pricing (expected this week)
- **Pricing Challenge:** Variable costs make per-API pricing difficult
  - Network bandwidth varies by API usage pattern
  - Anti-bot measures add variable costs
  - Site-specific factors (captcha frequency, response size)
  - Same API can cost 100x more depending on query

**Recent Developments:**
- Released biggest update to date this past week
- Still working out kinks post-update
- Sponsored Carnegie Mellon hackathon (recruiting focus)

**Team:**
- Has engineer on O1 visa (hackathon judging for immigration points)
- Recruiting friend from NYC (friend has free housing in Lower East Side)
- Small team (Alex handles most conversations)

## Strategic Fit Analysis

### Market Alignment
**Complementary ICPs:** Both Parse and Zo target non-technical users becoming technical

### Product Synergy
| Parse Provides | Zo Provides | Combined Value |
|----------------|-------------|----------------|
| Raw web data via API | Reasoning layer | Intelligent data processing |
| One-time or recurring data pulls | Persistent workflows | Automated data pipelines |
| Access to previously un-scrapable sites | Logic and rules | Smart data extraction |

### Business Model Alignment
- Parse moving to usage-based (from monthly credits)
- Zo already on usage-based
- Synergy: More Parse usage = More Zo usage = Revenue alignment

### Community Opportunity
- Zo Discord = 1,000 potential Parse customers
- Authentic engagement vs. marketing outreach
- Parse can contribute technical expertise to Zo community
