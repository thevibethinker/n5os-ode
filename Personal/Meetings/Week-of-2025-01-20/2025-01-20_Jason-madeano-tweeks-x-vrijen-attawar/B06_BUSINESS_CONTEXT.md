- **Current Status:** Post-pivot, building Tweeks as primary product
- **Stage:** Early-stage startup, post-YC

### Product: Tweeks
**What it is:** Generative UI engine / browser extension that allows users to modify any website's UI through natural language prompts

**How it works:**
1. Extracts page content from the website
2. Passes content to AI (Claude, similar LLMs)
3. Makes surgical modifications to specific elements based on user instructions
4. Re-renders the page with modifications applied while preserving everything else

**Product Variants:**
- **Consumer (Current Focus):** Chrome browser extension for end users to modify websites they visit
- **B2B (Future/Exploratory):** Direct integration into company websites without extension; can spin up PRs or custom builds for individual users

**Key Features:**
- Natural language prompts to modify UI
- Selector tool to identify specific page elements
- Library of shared tweaks (user-generated content)
- Shareable public links for individual tweaks
- "Not interested" button to remove modifications
