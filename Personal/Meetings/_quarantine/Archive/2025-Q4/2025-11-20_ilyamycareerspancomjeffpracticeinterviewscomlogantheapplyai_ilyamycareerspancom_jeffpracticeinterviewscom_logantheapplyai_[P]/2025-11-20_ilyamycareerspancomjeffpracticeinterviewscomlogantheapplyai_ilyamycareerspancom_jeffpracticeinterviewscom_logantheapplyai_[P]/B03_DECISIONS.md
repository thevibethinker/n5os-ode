# Decisions & Commitments

## APPROVED DECISIONS

### 1. Admin Access Model ✓
**Decision:** All three team members (Ilya, Jeff, Logan) will be granted admin access to Practice Interviews platform.

**Implication:** No single person becomes a blocker for user additions. Each can independently manage Careerspan user access to the tool.

**Status:** Executed during call — all three made admins

---

### 2. Engagement-Gated Access Model ✓
**Decision:** Practice Interviews access will NOT be open to all Careerspan users. Instead, access will be gated behind a quality-engagement barrier: users must complete 3+ career stories before unlocking Practice Interviews access.

**Rationale:** 
- "Action speaks louder than words" — users who have taken action to write career stories are signal-rich (motivated, engaged)
- Creates exclusivity factor: "Partnership only available to our most engaged users"
- Mirrors proven engagement models (free throw practice → better game performance)
- Avoids low-signal user noise
- Provides better data for partnership ROI analysis

**Alternative Rejected:** Open access to all Careerspan users ("just let's see what happens")

**Outcome:** Partnership quality and engagement metrics will be measurably better with gating

---

### 3. User Onboarding Approach ✓
**Decision:** Start with manual user addition process (admin adds users by email). Do NOT implement bulk upload upfront.

**Rationale:** "Start simple, right?" — manual process is lower friction to implement and test

**Future Path:** Bulk upload remains available if volume increases

---

### 4. User Engagement Tracking Framework ✓
**Decision:** Use admin dashboard analytics to track adoption and engagement metrics on Careerspan user cohort

**Key Metrics to Monitor:**
- Number of Practice Interviews questions attempted
- Monthly average scores
- Interview Academy completion % (learning mindset indicator)
- Engagement patterns (peak usage days, frequency)
- Quality signals: behavioral vs. hypothetical performance split

**Purpose:** Real-time feedback on partnership viability and user-product fit
