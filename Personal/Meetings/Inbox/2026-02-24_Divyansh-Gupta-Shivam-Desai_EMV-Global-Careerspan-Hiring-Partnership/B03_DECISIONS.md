# B03_DECISIONS

## Decision 1: Start with a Manual Joint Workflow

**DECISION:** EMV will first share candidate profiles with Careerspan, Careerspan will run cultural-fit assessment, and only qualified candidates will be passed back for client submission.  
**CONTEXT:** EMV already performs technical vetting; the gap identified was startup/founder-specific cultural fit.  
**DECIDED BY:** Divyansh Gupta and Vrijen Attawar, with Shivam Desai confirming alignment.  
**IMPLICATIONS:** The near-term process becomes a two-stage filter: technical evaluation (EMV) plus cultural-fit evaluation (Careerspan).  
**ALTERNATIVES CONSIDERED:** Direct client submission after only technical vetting.

## Decision 2: Careerspan Access for EMV Trial

**DECISION:** Vrijen will provide Divyansh access to Careerspan so EMV can evaluate and begin using the platform.  
**CONTEXT:** Divyansh requested practical understanding and workflow clarity before deeper integration.  
**DECIDED BY:** Vrijen Attawar (offering access) and Divyansh Gupta (accepting).  
**IMPLICATIONS:** Collaboration can begin immediately without waiting for technical integration work.  
**ALTERNATIVES CONSIDERED:** Waiting until API integration is complete before any usage.

## Decision 3: API Integration Deferred to a Later Phase

**DECISION:** API integration between EMV’s platform and Careerspan is agreed in principle but postponed until EMV’s current feature work is complete.  
**CONTEXT:** Divyansh noted his tech team is already committed to the next upcoming feature.  
**DECIDED BY:** Divyansh Gupta and Vrijen Attawar (mutual agreement), acknowledged by Shivam Desai.  
**IMPLICATIONS:** Operations proceed manually now; automation is planned as a later upgrade.  
**ALTERNATIVES CONSIDERED:** Immediate API build now.
