## Key Dependencies & Risks

### Critical Dependencies

1. **Aspire can recruit 1,000 alumni** - If recruitment stalls at 500, pilot viability reduced by 50%
   - Contingency: Phase gate at Week 2; if <500 recruited, discuss scaled-down version (500 users instead of 1K)

2. **Community manager availability** - Single point of failure for engagement programming
   - Contingency: Aspire identifies backup community manager by Week 0

3. **Hiring manager participation** - If no employers show up for AMAs, retention drops
   - Contingency: Vrijen + Husain jointly recruit managers from existing networks; offer incentive (e.g., free Careerspan access for their team)

4. **Careerspan can deliver weekly metrics** - Without real-time data, gamification/recognition breaks
   - Contingency: Vrijen builds lightweight dashboard API; fallback to manual exports if needed

### Risk Mitigation Playbook

| Risk | Early Warning Sign | Mitigation |
|---|---|---|
| **Phase 1 activation low** | <50% by Week 2 | Double down on Week 3-4 marketing; extend onboarding webinar; remove barriers |
| **Retention cliff at Week 5** | 30%+ drop in active users after Week 4 | Immediate community manager intervention; accelerate first AMA; emergency challenge |
| **Hiring manager AMA no-show** | <10 alumni register for first AMA | Vrijen personally recruits; offer Careerspan free trial for their hiring team |
| **Token costs exceed budget** | By 20%+ in first month | Pause new user onboarding; optimize existing user prompts; communicate with Aspire on cost-share adjustment |

## Success Scenarios (3 Months Forward)

### Scenario A: Strong Performance (Best Case)
- **Indicators:** 75%+ activation, 45%+ retention, 10+ hiring managers engaged, strong survey feedback
- **Implication:** Scale to 5,000 Aspire alumni + recruit 2-3 new partners (boot camps, community colleges)
- **Revenue signal:** 5,000 alumni × $1-2/mo token cost = $5-10K/mo infrastructure costs; offset by future employer contracts

### Scenario B: Moderate Performance (Acceptable)
- **Indicators:** 70%+ activation, 38%+ retention, 4-5 hiring managers engaged, mixed feedback
- **Implication:** Continue 6-month pilot; double down on Phase 2 engagement; plan iteration for Year 2
- **Revenue signal:** Proof of concept validated; runway for fundraising (if desired) or organic scaling

### Scenario C: Weak Performance (Pivot Required)
- **Indicators:** <65% activation, <35% retention, 2 hiring managers engaged, low survey scores
- **Implication:** Pause program; conduct root cause analysis (product? engagement model? recruitment issue?); redesign for Year 2
- **Revenue signal:** B2B model viability questioned; revert to organic growth + targeted employer pilots

## Final Note: Why This Matters

This isn't just a "partnership." It's a **business model validation experiment** for Careerspan.

Success criteria:
1. ✓ Can engagement programming (community + challenges + accountability) drive retention from 6% → 40%?
2. ✓ Can employers (hiring managers) see measurable value in AI-powered candidate narratives?
3. ✓ Can Careerspan scale B2B via partner ecosystem (Aspire, boot camps, etc.) rather than direct sales?

The next 6 months will answer these questions. Everything after (pricing, funding decisions, hiring roadmap) depends on these outcomes.
