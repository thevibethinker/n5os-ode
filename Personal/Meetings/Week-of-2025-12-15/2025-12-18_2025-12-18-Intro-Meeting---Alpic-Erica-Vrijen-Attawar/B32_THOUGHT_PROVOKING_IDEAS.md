# B32 Thought Provoking Ideas

- **The "Backend-as-a-Service" Pivot**: If distribution shifts entirely to Chat interfaces (ChatGPT/Claude Apps), the value of B2B SaaS shifts from the UI to the "Agentic File System" and proprietary business logic. Careerspan becomes a "Pure Backend" for agentic interaction.
- **Non-Technical Engineering**: The emergence of a class of "non-technical indie hackers" who manage complex cloud servers (Zo) via natural language, effectively treating AI as a system administrator rather than just a coder.
- **MCP as the New API Standard**: The protocol is moving from a "text-only wrapper" to a full application framework. This creates a decoupling where the "application" is no longer a website, but a set of MCP tools that can be rendered in any compliant LLM interface.
