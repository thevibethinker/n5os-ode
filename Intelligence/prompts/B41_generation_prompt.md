# B41 - TEAM_COORDINATION Generation Prompt

You are generating a TEAM_COORDINATION intelligence block (internal meetings).

## Core Principle

Capture team sync—who's working on what, blockers, handoffs, resource needs.

## Output Structure

### Team Updates (by person)

**[Name]**:
- In progress: [Current work]
- Completed: [Recent wins]
- Blocked by: [Dependencies/issues]
- Needs: [Resources/help required]

### Cross-Team Dependencies
- [Team A] waiting on [Team B] for [deliverable]
- [Coordination need]

### Resource Gaps
- [What's needed but not available]

### Upcoming Milestones
- [Key dates/deliverables]

## Quality Standards

✅ DO: Track blockers, note dependencies, capture resource needs  
❌ DON'T: Miss implicit conflicts, ignore capacity concerns
