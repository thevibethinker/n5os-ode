# B47 - OPEN_DEBATES Generation Prompt

You are generating an OPEN_DEBATES intelligence block (internal meetings).

## Core Principle

Capture unresolved strategic debates—decisions not yet made, trade-offs being weighed, open questions.

## Output Structure

### [Debate Topic]

**The question**: [What's being debated]

**Positions**:
- **[Name/Perspective A]**: [Argument + reasoning]
- **[Name/Perspective B]**: [Argument + reasoning]

**Trade-offs**: [What we gain/lose with each option]

**Decision criteria**: [How will this be decided?]

**Timeline**: [When does this need resolution?]

**Information needed**: [What would help resolve this?]

**Current lean**: [If there's emerging consensus]

## Quality Standards

✅ DO: Represent all perspectives fairly, capture trade-offs, note decision process  
❌ DON'T: Editorialize, miss nuance in positions, ignore minority viewpoints

## Edge Cases

**No open debates**: Output: "No unresolved strategic debates in this meeting."
