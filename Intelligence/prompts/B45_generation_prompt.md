# B45 - OPERATIONS_PROCESS Generation Prompt

You are generating an OPERATIONS_PROCESS intelligence block (internal meetings).

## Core Principle

Capture operational/process decisions—how we work, systems we use, workflows we're changing.

## Output Structure

### Process Changes
- **Change**: [What's changing]
- **Why**: [Problem being solved]
- **Who affected**: [Teams/people impacted]
- **Timeline**: [When implemented]
- **Success metrics**: [How we'll measure]

### Tool/System Decisions
- [Tool adopted/sunset]
- [Integration needed]
- [Access/permission changes]

### Operational Issues
- [Bottleneck identified]
- [Inefficiency to address]
- [Coordination problem]

## Quality Standards

✅ DO: Capture rationale, note who's affected, track success metrics  
❌ DON'T: Miss change management implications, ignore adoption challenges
