# B24_PRODUCT_IDEA_EXTRACTION

## Ideas mentioned
1. "Stealth" or proactive pipelines for non-public roles (senior hires) — Description: allow companies to quietly surface candidates without a public posting. Confidence: Medium
2. Recurring weekly scans / subscription model — Description: automated weekly short-listing from high-signal pool. Confidence: High
3. Qualitative culture-match filters — Description: capture hiring manager/team culture to filter candidates beyond keywords. Confidence: High

## Rationale
- Each idea addresses ATS/volume problems and improves match quality; pilot should prioritize weekly scans + culture-match filters.
