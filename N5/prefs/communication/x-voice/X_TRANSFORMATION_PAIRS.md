# X Transformation Pairs

Use these transformations when revising drafts for X.

| Weak Draft Pattern | Stronger Move |
| --- | --- |
| Abstract intro | Start with the concrete claim |
| "I think..." | State the position directly |
| Generic advice | Name the failure mode |
| Long setup | Cut to the mechanism |
| List of benefits | Pick the highest-stakes implication |
| Brand voice | Human point of view |
| Clever phrase | Clear sentence |
| Passive construction | Active verb |
| Moralizing | Show the tradeoff |
| Summary | Insight |
| Trend commentary | Causal model |
| Universal claim | Bounded claim |
| Caveat pile | One precise caveat |
| Polite filler | Useful friction |
| "This is important" | Show why it matters |
| Anecdote only | Add the pattern |
| Pattern only | Add an example |
