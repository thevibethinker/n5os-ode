# Transformation Pairs Library

**Purpose:** Few-shot examples for voice conditioning. The LLM learns V's voice by studying the transformation from neutral → V-voice.

**Usage:** Reference specific pairs by tag when generating content. Platform-specific pairs in `platforms/x.md` and `platforms/linkedin.md`.

---

## Email: Warm Intro

**Tags:** `email`, `intro`, `warm`, `connector`

**Style-Free:**
> Connecting Jabari with Ben Guo (Zo). Reason: alignment between AI Collective and Zo. Notable facts: Ben early at Venmo, ~10yrs at Stripe, Zo = promptable cloud computer. My POV: use it daily; built personal OS on it.

**V-Voice:**
> Hi Jabari — … one of the founders of Zo Computer. … I've built my entire personal operating system on it — 250k+ lines of code … It's essentially a computer in the cloud you can prompt and set to work on your behalf.

**Pattern:** Concrete pedigree → lived-use proof → vivid descriptor → one clear ask

## Email: Follow-Up After Event

**Tags:** `email`, `follow-up`, `event`, `apology`

**Style-Free:**
> Apology for delay due to family emergencies. Provide update. Request: sponsor if possible; otherwise, confirm attendance.

**V-Voice:**
> Apologies for going quiet … back-to-back family emergencies … Thankfully, things have settled. … I realize it's likely too late to secure sponsorships … If not, I'll aim to make it back on track.

**Pattern:** Human context first → then agenda → specific constraints → polite but direct

## Email: Meeting Request

**Tags:** `email`, `meeting`, `ask`, `direct`

**Style-Free:**
> Hi! I hope you're doing well. I wanted to reach out and see if you might have some time over the next week or so to potentially chat about the project? No rush at all.

**V-Voice:**
> Can we do 20 minutes Thursday to discuss the project?

**Pattern:** Kill qualifiers → Name the ask → Propose specific time

## Email: Follow-Up Bump

**Tags:** `email`, `follow-up`, `bump`, `deadline`

**Style-Free:**

**V-Voice:**

## Quick Reference: Transformation Heuristics

| Input Signal | → Transformation |
|--------------|------------------|
| "I was wondering if..." | [Just ask] |
| "I think maybe we could..." | "Let's..." or "Can we..." |
| "Just wanted to check in" | "Where are we on [X]?" |
| "No rush, but..." | [Give specific deadline] |
| "If you have time..." | [Propose specific time] |
| "Does that make sense?" | [End with confidence] |
| "Let me know what you think" | "Let me know by [date]" |
| "Sorry to bother you" | [Delete — just ask] |
