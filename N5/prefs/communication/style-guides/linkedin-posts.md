# LinkedIn Posts — Style Guide

Scopes:
- Thought leadership
- Vulnerable origin story
- Rant / edge take
- Milestone announcement
- Product explainer

General voice:
- Conversational, deliberate, human
- Specificity over abstractions
- Rhythmic variety: mix flowing paragraphs with occasional one-liners
- Emojis as structural markers only (sparingly)

Universal structure (hide the scaffolding):
- Hook that earns attention without clickbait
- 2–3 concrete beats (stories, numbers, details)
- One insight or turn
- Soft CTA (comment prompt, invite DM) when relevant

Style-free → V workflow:
1) Draft style-free version: just facts / beats / claims
2) Layer V cadence: specific nouns, human aside, remove performative vulnerability
3) Read aloud test: should feel like a conversation

Patterns by type:

1) Thought leadership
- Start with a contrarian or clarifying statement
- Give one example from your work (Careerspan/N5)
- Offer a simple framework or lens
- CTA: “Curious how others handle this.”

2) Vulnerable origin story
- One true moment (context switching, not following up, feeling behind)
- One comparison that stung (others who “have it together”)
- What changed (N5OS as agency)
- Today’s practice (one concrete habit)

3) Rant / edge take
- Start with the absurdity (founder math, etc.)
- 3 receipts (numbers, observed behaviors)
- Turn: what actually matters
- CTA: light, invite counterexamples

4) Milestone announcement
- Lead with the thing (no adjectives)
- Why it matters for others
- Gratitude to specific humans (1–2 names ok if public)
- Link / next step

5) Product explainer
- Problem in one line (real, felt)
- What we built + for whom
- Mechanism, not magic (how it works)
- Example in the wild

QA checklist (all types):
- Can a single paragraph be read aloud naturally?
- Are there at least 2 specifics (numbers, names, places)?
- Zero cliches / stop verbs
- Would you actually post this as-is?
