# Style Guides (Operational)

Purpose: channel- and format-specific guidance that turns the core voice into repeatable outputs. Each guide includes structure, tone, lexicon, templates, and a QA checklist.

This layer sits on top of:
- Core voice → `file 'N5/prefs/communication/voice.md'`
- Social voice → `file 'N5/prefs/communication/social-media-voice.md'`
- Voice routing → `file 'N5/prefs/communication/voice-routing-rules.md'`
- Transformation method → `file 'N5/prefs/communication/voice-transformation-system.md'`

Available guides:
- Blurbs → `file 'N5/prefs/communication/style-guides/blurbs.md'`
- Follow-up emails → `file 'N5/prefs/communication/style-guides/follow-up-emails.md'`
- LinkedIn posts → `file 'N5/prefs/communication/style-guides/linkedin-posts.md'`
- Transformation Pairs Library (seed) → `file 'N5/prefs/communication/style-guides/transformation-pairs-library.md'`

How to use (quick):
1) Identify the content type and open its guide.
2) If available, pull 2–3 matching examples from the Transformation Pairs Library.
3) Draft using the guide’s structure + templates.
4) Run the QA checklist before sharing/posting.
