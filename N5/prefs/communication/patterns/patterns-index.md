# Pattern Index

Tags: hook, specificity, rhythm, proof, contrast, CTA, vulnerability, narrative, list, analogy

- Contrast Hook (hook, contrast)
  - Effect: Snap attention via unexpected juxtaposition
  - Use: Openers for LI posts, blurbs
  - How: A vs B in one line; immediately ground with a concrete example
  - Example: "Founder math is wild." → money list → principle
  - Anti: Don’t stack 3+ contrasts; avoid cliche oppositions

- Identity Flip (hook, narrative, vulnerability)
  - Effect: Humanizes by revealing a reversal of role/identity
  - Use: Origin stories, vulnerable posts
  - How: Past identity → present tension → one lived specific → resolution
  - Example: "I spent 10 years giving career advice. Then I became a founder …"
  - Anti: Don’t moralize; keep resolution modest, not heroic

- Two Specifics Rule (specificity)
  - Effect: Increases credibility and texture
  - Use: All formats
  - How: Include at least two of: numbers, names, places, timeframes, artifacts
  - Example: "250k+ lines, Stripe decade"
  - Anti: Don’t dump stats; pick the two that matter

- List Cadence (rhythm)
  - Effect: Builds momentum without sounding like a thread farm
  - Use: Rants, product explainers
  - How: 3-beat list with rising stakes; vary sentence length
  - Example: $8K tools → $25K booth → $50K fractional …
  - Anti: Don’t use single-sentence paragraphs throughout

- Show-Then-Name (proof)
  - Effect: Demonstrates before declaring
  - Use: Product, case studies, blurbs
  - How: Give concrete behavior first, then label the capability
  - Example: "Hit APIs, spin local LLMs … It’s a promptable cloud computer"
  - Anti: Don’t lead with abstraction

- Gentle Ask (CTA)
  - Effect: Invites without performative engagement
  - Use: LI thought-leadership, emails
  - How: One question or next step + honest context
  - Example: "Curious how others handle this."
  - Anti: Avoid “Like/comment/DM” bundles

- Human Context First (narrative)
  - Effect: Builds trust and reduces friction
  - Use: Follow-ups
  - How: State the human factor briefly before the ask
  - Example: "Back-to-back family emergencies … now focused on X"
  - Anti: Don’t overshare or dramatize

- Visual Money (specificity, hook)
  - Effect: Makes abstract prioritization tangible
  - Use: Rants about spending, tradeoffs
  - How: Use real dollar figures in ascending order
  - Example: $8K → $25K → $50K
  - Anti: Don’t invent numbers; keep plausible

- One Human Detail (specificity)
  - Effect: Warms tone without fluff
  - Use: Blurbs, bios
  - How: Include a single concrete human detail
  - Example: "lives in it daily"
  - Anti: Don’t stack three “quirks”

- Read-Aloud Test (rhythm)
  - Effect: Exposes robotic cadence
  - Use: Final pass on any piece
  - How: Read once aloud; fix any clunk or breathless run
  - Anti: Don’t optimize for “engagement hacks” over clarity
