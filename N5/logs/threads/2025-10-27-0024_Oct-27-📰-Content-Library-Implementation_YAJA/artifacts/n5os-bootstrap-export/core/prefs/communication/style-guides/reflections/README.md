# Reflection Style Guides

**Purpose:** Block-specific style guides for reflection processing (B50-B99)

## Block Types

- **B50**: Personal Reflection - Stream-of-consciousness self-reflection
- **B60**: Learning & Synthesis - Insights from research/reading/conversations  
- **B70**: Thought Leadership - Original thinking for professional audience
- **B71**: Market Analysis - Market trends and competitive landscape
- **B72**: Product Analysis - Product decisions and roadmap
- **B73**: Strategic Thinking - High-level strategy and vision
- **B80**: LinkedIn Post - Social content for professional audience
- **B81**: Blog Post - Long-form content for blog/publication
- **B82**: Executive Memo - Structured memos for stakeholders
- **B90**: Insight Compounding - Pattern synthesis across reflections
- **B91**: Meta-Reflection - Reflection on reflection process

## Style Guide Development

Each style guide should define:
1. **Voice & Tone**: Formality, emotional register, perspective
2. **Structure**: Expected sections, flow, organization
3. **Length**: Target word count ranges
4. **Examples**: Sample outputs demonstrating style
5. **Anti-patterns**: What to avoid
