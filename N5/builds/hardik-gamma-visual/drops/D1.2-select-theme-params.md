# D1.2: Select Theme and Visual Parameters

## Objective

Determine optimal Gamma parameters for a professional candidate assessment webpage.

## Tasks

### 1. List Available Themes
Run: `bun run Skills/gamma/scripts/gamma.ts themes`

### 2. Select Theme
Choose a theme that conveys:
- Professional/corporate credibility
- Clean data visualization
- Trust and authority
- NOT overly playful or casual

### 3. Determine Parameters

Consider and decide on:

| Parameter | Options | Recommendation |
|-----------|---------|----------------|
| `--format` | webpage | **webpage** (required) |
| `--mode` | generate, condense, preserve | **generate** (let AI structure) |
| `--tone` | professional, analytical, etc. | **"professional and analytical"** |
| `--audience` | description | **"startup founders evaluating engineering candidates"** |
| `--amount` | brief, medium, detailed | **medium** (enough detail, not overwhelming) |
| `--images` | aiGenerated, webSearch, none | **none** or **aiGenerated** with subtle style |
| `--image-style` | if using images | **"minimal abstract professional"** |
| `--visibility` | private, public, unlisted | **unlisted** (shareable but not indexed) |
| `--cards` | number | **8-12** (hero + 6 evidence + gaps + risk + questions + matrix) |

### 4. Image Decision

Evaluate whether images add value:
- **Pro:** Visual appeal, breaks up text
- **Con:** AI-generated images may look generic, distract from data
- **Recommendation:** Start with `--images none` for data-focused presentation. Can regenerate with images if needed.
