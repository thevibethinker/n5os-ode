# D2.1: Generate Gamma Webpage

## Objective

Use the Gamma API to generate the candidate profile webpage.

## Prerequisites

Read these artifacts from Stream 1:
- `N5/builds/hardik-gamma-visual/artifacts/gamma_input.md` (content)
- `N5/builds/hardik-gamma-visual/artifacts/gamma_params.json` (parameters)

## Execution

### 1. Construct Command

Build the Gamma generate command using the params:

```bash
bun run Skills/gamma/scripts/gamma.ts generate "$(cat N5/builds/hardik-gamma-visual/artifacts/gamma_input.md)" \
  --mode <from params> \
  --format webpage \
  --theme <theme_id from params> \
  --tone "<tone from params>" \
  --audience "<audience from params>" \
  --amount <amount from params> \
  --images <images from params> \
  --visibility unlisted \
  --cards <cards from params> \
  --wait
```

### 2. Run Generation

Execute the command. The `--wait` flag will poll until complete.

### 3. Capture Output

The command will return:
- `gammaUrl` — The public URL to the generated webpage
- `generationId` — For status checks if needed

### 4. Verify Generation

Open the URL and confirm:
- Page loads correctly
- All sections are present
- No obvious rendering errors

## Output

Write URL to:
`N5/builds/hardik-gamma-visual/artifacts/gamma_url.txt`

Format:
```
url: <gamma webpage URL>
generation_id: <generation ID>
generated_at: <ISO timestamp>
```
