# Worker Brief: Edge-to-Edge Fit + Responsive Tuning

## Context

W2.1 successfully implemented pinned concept grid layout. Concepts now distribute as hubs across the canvas. However:

1. **Edge-to-edge fit is not optimal** - graph doesn't fill the available viewport properly
2. **Grid spacing is hardcoded** - 300px × 250px doesn't adapt to different viewport sizes
3. **zoomToFit padding** may be too large (currently 10)

## Your Task

Fix the frame/viewport fit and make the layout responsive.
