# D3.1: Wire Task System into Thread-Close

## Objective
Update the thread-close skill to properly integrate with task-system using the new context-based approach.

## Context
Current thread-close has task integration documented in SKILL.md but relies on old `N5/task_system/close_hooks.py` with regex. We need to:
1. Update thread-close to call task-system's context gatherers
2. Remove dependency on old close_hooks.py
3. Ensure the AI reasoning flow is clear

## Tasks

1. **Update `Skills/thread-close/SKILL.md`** Task System Integration section:

Replace the current integration docs with:
```markdown
### Task System Integration

When closing a conversation, check if it's an action conversation:

1. **Check for action conversation:**
   ```bash
   python3 Skills/task-system/scripts/context.py action-check --convo-id <id>
   ```

2. **If task found**, get completion context:
   ```bash
   python3 Skills/task-system/scripts/context.py completion-check --convo-id <id> --task-id <id>
   ```

3. **Reason about completion** using the context (see task-system SKILL.md)

4. **Show assessment to V:**
   ```
   TASK: Draft investor memo
   
   Assessment: ✅ COMPLETE
   Evidence: investor-memo-v1.md created, matches task intent
   
   Mark as: [complete] [partial] [blocked]
   ```

5. **Update task** based on V's choice:
   ```bash
   python3 Skills/task-system/scripts/task.py complete <id>
   # or
   python3 Skills/task-system/scripts/task.py update <id> --status in_progress
   ```

6. **If partial, get next step:**
   ```bash
   python3 Skills/task-system/scripts/context.py next-step --task-id <id>
   ```
   Show V the suggested follow-up.
```

2. **Update `N5/lib/close/core.py`** (or create new integration module):
   - Remove imports from `N5.task_system.close_hooks`
   - Add function to call task-system context scripts
   - Return context in `gather_thread_context()` output

3. **Create integration test scenario** in thread-close:
   - Document the full flow from close → task check → assessment → update

4. **Deprecate old files:**
   - Add deprecation notice to `N5/task_system/close_hooks.py`
   - Add deprecation notice to `N5/task_system/action_tagger.py`
   - Point to new Skill location

## Key Change
**Before:** close_hooks.py used regex to assess completion
**After:** context.py gathers context, AI reasons, task.py updates

## Output
- Updated `Skills/thread-close/SKILL.md`
- Updated `N5/lib/close/core.py` 
- Deprecation notices on old files
