# W3.2 — Execute Dedupe Migration (IMPLEMENTATION)

## Goal
Execute the 8-step migration plan from W2.2 to dedupe items, create redirects, retrofit subtypes, and deprecate orphans.

## Dependency
**Requires W3.1 complete** — needs `subtype` column and `id_redirects` table to exist.

## Context from W2.2
Found these duplicates by URL:
- 8 Calendly duplicates: `calendly_*` (new) vs `meeting_booking_*` (legacy)
- 1 Notion duplicate: `case_notes_mckinsey` vs `guides_mckinsey_consulting_mba`
- 5 orphan `vrijen_*` items with no URL (to deprecate)

### 2. Execute 8-step migration (per W2.2)

**Step 1: Backup**
```bash
cp N5/data/content_library.db N5/data/backups/content_library_pre_v5_$(date +%Y%m%d_%H%M%S).db
```

**Step 2: Identify duplicates by URL**
```sql
SELECT url, GROUP_CONCAT(id, ', ') as ids, COUNT(*) as cnt
FROM items 
WHERE url IS NOT NULL AND url != ''
GROUP BY url 
HAVING cnt > 1;
```

**Step 3: For each duplicate set, determine canonical ID**
- Calendly items: `calendly_*` is canonical (tied to active sync)
- Others: prefer newer ID or more complete record

**Step 4: Merge curated tags from legacy → canonical**
```sql
-- For each (old_id, new_id) pair:
INSERT OR IGNORE INTO tags (item_id, tag_key, tag_value)
SELECT new_id, tag_key, tag_value 
FROM tags 
WHERE item_id = old_id
AND NOT EXISTS (
    SELECT 1 FROM tags WHERE item_id = new_id AND tag_key = tags.tag_key
);
```

**Step 5: Create redirects**
```sql
INSERT INTO id_redirects (old_id, new_id) VALUES (?, ?);
```

**Step 6: Deprecate legacy duplicates**
```sql
UPDATE items SET deprecated = 1 WHERE id IN (...old_ids...);
```

**Step 7: Deprecate orphan vrijen_* items**
```sql
UPDATE items SET deprecated = 1 WHERE id LIKE 'vrijen_%' AND url IS NULL;
```

**Step 8: Backfill subtypes**
```sql
-- Calendly scheduling links
UPDATE items SET subtype = 'scheduling-link' 
WHERE content_type = 'link' 
AND (source = 'calendly-sync' OR source_file_path LIKE '%calendly%');

-- Profile links (LinkedIn, personal sites)
UPDATE items SET subtype = 'profile' 
WHERE content_type = 'link' 
AND (url LIKE '%linkedin.com%' OR url LIKE '%vrijenattawar.com%' OR url LIKE '%github.com/%');
```
