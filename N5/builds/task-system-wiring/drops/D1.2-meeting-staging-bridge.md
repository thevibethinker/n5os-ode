# D1.2: Meeting to Staging Bridge

## Objective
Create the bridge that takes processed meeting action items (B05 blocks) and stages them for review before they enter the task registry.

## Context
Flow:
```
Meeting folder [R] state
    │
    ▼
B05_ACTION_ITEMS.md (decomposed format from D1.1)
    │
    ▼
meeting_staging_bridge.py (THIS DROP)
    │
    ▼
staging.db (pre_task_staging table)
    │
    ▼
Evening accountability shows staged items for review
    │
    ▼
V approves/dismisses → approved items → task registry
```

## Existing Infrastructure

**Staging system**: `/home/workspace/N5/task_system/staging.py`
- `stage_potential_task()` — adds item to staging
- `get_pending_staged_tasks()` — retrieves pending items
- `promote_staged_task()` — moves to task registry
- `dismiss_staged_task()` — removes from staging

**Meeting folders**: `/home/workspace/Personal/Meetings/`
- Format: `Week-of-YYYY-MM-DD/YYYY-MM-DD_Name/`
- State markers: `[M]`, `[P]`, `[R]`
- B05 file: `B05_ACTION_ITEMS.md`

#!/usr/bin/env python3
"""
Meeting to Staging Bridge

Scans processed meetings ([R] state) for B05 action items
and stages them for review.
"""

from pathlib import Path
from typing import List, Dict, Optional
from datetime import datetime

# Import from existing modules
from N5.task_system.staging import stage_potential_task
from N5.task_system.b05_parser import parse_b05_structured  # From D1.1

MEETINGS_ROOT = Path("/home/workspace/Personal/Meetings")

def find_unprocessed_meetings() -> List[Path]:
    """
    Find meeting folders in [R] state that haven't had
    their action items staged yet.
    
    Check for marker file: .action_items_staged
    """
    pass

def process_meeting_action_items(meeting_folder: Path) -> List[int]:
    """
    Extract action items from B05 and stage them.
    
    Returns list of staged_task IDs.
    """
    b05_path = meeting_folder / "B05_ACTION_ITEMS.md"
    if not b05_path.exists():
        return []
    
    # Parse B05 (using D1.1's parser)
    items = parse_b05_structured(b05_path.read_text())
    
    staged_ids = []
    for item in items:
        # Stage each item
        staged_id = stage_potential_task(
            title=item['title'],
            source_type='meeting',
            source_id=meeting_folder.name,
            context=item.get('raw_text', ''),
            suggestions={
                'priority_bucket': item.get('priority_bucket', 'normal'),
                'domain_name': item.get('domain_name'),
                'project_name': item.get('project_name'),
                'estimated_minutes': item.get('estimated_minutes'),
                'first_step': item.get('first_step'),
                'milestones': item.get('milestones', [])
            }
        )
        staged_ids.append(staged_id)
    
    # Mark meeting as processed
    (meeting_folder / '.action_items_staged').write_text(
        datetime.now().isoformat()
    )
    
    return staged_ids

def process_all_pending_meetings() -> Dict[str, List[int]]:
    """
    Process all meetings with unstaged action items.
    
    Returns dict of meeting_folder -> staged_ids
    """
    results = {}
    for meeting_folder in find_unprocessed_meetings():
        staged_ids = process_meeting_action_items(meeting_folder)
        if staged_ids:
            results[meeting_folder.name] = staged_ids
    return results
