# D2.2: CLI Cart Commands

## Objective
Implement CLI commands for Content Carts in the `n5 content` namespace.

## Context
Building on D1.2's schema, we need user-facing CLI commands to manage content carts.

## Commands to Implement

### n5 content cart create <name> [--description "..."]
Create a new content cart.
```bash
n5 content cart create "hiring-intel-research" --description "Articles and papers on AI hiring"
# Output: Cart created: cart-xyz-789
```

### n5 content cart list
List all carts (active first, then archived).
```bash
n5 content cart list
# Output:
# ACTIVE:
#   cart-xyz-789: hiring-intel-research (5 items)
#   cart-xyz-790: interview-prep (3 items)
# ARCHIVED:
#   cart-xyz-100: old-project (12 items)
```

### n5 content cart show <cart_id>
Show cart contents with item titles and notes.
```bash
n5 content cart show "cart-xyz-789"
# Output:
# Cart: hiring-intel-research
# Description: Articles and papers on AI hiring
# Items (5):
#   1. [article] The Future of AI in Hiring
#      Note: Key source for argument
#   2. [paper] Bias in Algorithmic Hiring
#      Note: Counterpoint to consider
```

### n5 content cart add <cart_id> <item_id> [--note "..."]
Add an item to a cart.
```bash
n5 content cart add "cart-xyz-789" "article-123" --note "Key source for argument"
```

### n5 content cart remove <cart_id> <item_id>
Remove an item from a cart.

### n5 content cart search <cart_id> <query>
Search within a cart's items using existing content search.

### n5 content cart synthesize <cart_id> [--prompt "..."] [--output <file>]
The big one: synthesize all items in the cart.
```bash
n5 content cart synthesize "cart-xyz-789" --prompt "What are the common themes across these sources?" --output synthesis.md
# Output:
# Synthesizing 5 items...
# Generated synthesis with 3 themes
# Saved to: synthesis.md
```

### n5 content cart archive <cart_id>
Archive a cart (soft delete).

### n5 content cart delete <cart_id> [--force]
Hard delete a cart and all its items.

## Synthesis Implementation

The synthesize command should:
1. Fetch all items in the cart
2. Combine their content
3. Send to LLM with the prompt
4. Save result (to file or as a new content item)
