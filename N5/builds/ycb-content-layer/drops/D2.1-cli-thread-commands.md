# D2.1: CLI Thread Commands

## Objective
Implement CLI commands for Entry/Comment threading in the `n5 content` namespace.

## Context
Building on D1.1's schema, we need user-facing CLI commands to interact with threads and comments.

## Commands to Implement

### n5 content thread create <item_id> [--title "..."]
Create a new thread rooted at a content item.
```bash
n5 content thread create "abc-123" --title "Research on hiring intelligence"
# Output: Thread created: thread-xyz-789
```

### n5 content thread list <item_id>
List all threads for a content item.
```bash
n5 content thread list "abc-123"
# Output:
# thread-xyz-789: Research on hiring intelligence (3 comments)
# thread-xyz-790: Interview prep notes (1 comment)
```

### n5 content comment add <thread_id> [--parent <comment_id>] [--type note|synthesis|quote] <content>
Add a comment to a thread.
```bash
n5 content comment add "thread-xyz-789" --type synthesis "Key insight: AI-assisted hiring improves match quality by 40%"
```

### n5 content thread show <thread_id>
Display a thread with all comments in tree format.
```bash
n5 content thread show "thread-xyz-789"
# Output:
# Thread: Research on hiring intelligence
# Root: [article] The Future of AI in Hiring
# 
# ├── [synthesis] Key insight: AI-assisted hiring improves match quality by 40%
# │   └── [note] Need to verify this stat with original source
# └── [quote] "The best hires come from structured evaluation, not gut feel"
```

### n5 content comment edit <comment_id> <new_content>
Edit an existing comment.

### n5 content comment delete <comment_id>
Delete a comment (with confirmation).

## Implementation Notes

### File Location
`N5/scripts/n5_content_cli.py` - extend the existing content CLI or create new module.

### Tree Display Algorithm
For `thread show`, render comments as a tree:
```python
def render_tree(comments, parent_id=None, depth=0):
    for c in comments:
        if c['parent_comment_id'] == parent_id:
            indent = "  " * depth + ("└── " if depth > 0 else "")
            print(f"{indent}[{c['comment_type']}] {c['content'][:60]}...")
            render_tree(comments, c['id'], depth + 1)
```
