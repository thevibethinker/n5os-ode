# D2.3: Evening Accountability Agent

## Objective
Create a scheduled agent that texts V every evening with an accountability check and staged task review.

## Context
V's accountability system needs:
- Evening text reviewing what was accomplished vs. planned
- Honest reporting: "You committed to 5, completed 3"
- Review staged tasks from today (meetings, convos) — promote or dismiss
- Queue tomorrow's priorities
- Latency tracking: Record actual completion times

## Design

### Evening Flow (7:00 PM ET)

1. **Pull day's results**:
   - What was planned (from morning lock)
   - What was completed (task status = complete, completed_at = today)
   - What's still open

2. **Generate accountability report**:
   ```
   Evening check-in, V.
   
   TODAY'S SCORE: 3/5 tasks (60%)
   
   ✅ Completed:
   - Investor memo (sent 2:34 PM)
   - Invoice review
   - Logan follow-up
   
   ❌ Not done:
   - Reply to Sarah (now 3 days overdue)
   - Partnership proposal draft
   
   LATENCY: Avg 0.5 days ahead of due date 📈
   
   What happened with the incomplete tasks?
   ```

3. **Process response**: V explains or reschedules

4. **Staged task review**:
   ```
   NEW ITEMS captured today (3):
   
   From meeting with Logan:
   - [ ] Research competitor X pricing
   - [ ] Draft partnership template
   
   From conversation:
   - [ ] Send Heather the doc
   
   Reply with numbers to promote (e.g., "1,3") or "none"
   ```

5. **Queue tomorrow**: Ask what's priority for tomorrow

### Agent Configuration

```
RRULE: FREQ=DAILY;BYHOUR=19;BYMINUTE=0
Delivery: sms

Instruction:
[Agent prompt that executes the evening flow]
```
