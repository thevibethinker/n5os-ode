## Context

The build orchestrator needs standardized templates for:
1. **BUILD.md** — The living orchestrator document (different from PLAN.md)
2. **Worker briefs** — Self-contained documents V pastes into new threads
3. **meta.json schema** — Machine-readable build state

---

## Template 1: BUILD_template.md

This is the **orchestrator's living document** — different from PLAN.md (which is Architect's execution plan).

Sections:
- **Header**: Build title, objective, type, created date
- **Current Status**: Quick summary of where we are
- **Wave Progress**: Visual/table of wave completion
- **Worker Status Table**: Each worker with status, thread link, notes
- **Learnings Log**: Aggregated insights from worker completions
- **Blockers/Concerns**: Active issues needing attention
- **Next Actions**: What to do next (launch wave, review, close)

Include placeholders: `{{SLUG}}`, `{{TITLE}}`, `{{DATE}}`, etc.

## Template 3: meta_schema.json

Reference schema (not a template to fill, but a spec):

```json
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "type": "object",
  "required": ["slug", "title", "created", "status", "type"],
  "properties": {
    "schema_version": { "type": "string", "default": "1.0" },
    "slug": { "type": "string" },
    "title": { "type": "string" },
    "created": { "type": "string", "format": "date" },
    "status": { "enum": ["draft", "active", "complete", "abandoned"] },
    "completed_at": { "type": ["string", "null"] },
    "type": { "enum": ["code_build", "content", "research", "general"] },
    "orchestrator_convo_id": { "type": ["string", "null"] },
    "workers": {
      "type": "object",
      "properties": {
        "total": { "type": "integer" },
        "complete": { "type": "integer" },
        "in_progress": { "type": "integer" },
        "blocked": { "type": "integer" },
        "pending": { "type": "integer" }
      }
    },
    "waves": {
      "type": "array",
      "items": {
        "type": "object",
        "properties": {
          "wave": { "type": "integer" },
          "workers": { "type": "array", "items": { "type": "string" } },
          "status": { "enum": ["pending", "in_progress", "complete"] }
        }
      }
    }
  }
}
```
