## Steps

1. **Create staging directory**: `/home/workspace/N5/builds/career-coaching-hotline/artifacts/source-pdfs/`
2. **Download all 24 files** from Google Drive using `use_app_google_drive` tool with `google_drive-download-file` action. Download each file to the staging directory.
3. **Extract text from each PDF** using `read_file` (which can read PDFs natively). For the TXT file, just read it directly.
4. **Write extracted text** to markdown files in `/home/workspace/N5/builds/career-coaching-hotline/artifacts/extracted/` with frontmatter:
   ```yaml
   ---
   source_file: <original filename>
   source_id: <google drive file id>
   extracted: YYYY-MM-DD
   provenance: career-coaching-hotline/D1.1
   ---
   ```
5. **Create a manifest** at `/home/workspace/N5/builds/career-coaching-hotline/artifacts/extraction-manifest.json` listing all files, their extraction status, word counts, and detected topics.
