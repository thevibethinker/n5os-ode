# D5.1: Deploy + Service Registration + Documentation

## Objective

Deploy the Career Coaching Hotline as a registered Zo user service, create the Skill documentation, and prepare everything for V to create the VAPI assistant and Fillout form.

## Steps

### 1. Service Registration

Register the webhook server as a Zo user service:
- Check port registry: `python3 N5/scripts/port_registry.py next n5-services`
- Register: `register_user_service` with the assigned port
- Verify: `service_doctor` to confirm health

Naming convention: `career-coaching-hotline` (service label)

### 2. Update Port Registry

Add the new service to `N5/config/PORT_REGISTRY.md`.

### 3. SKILL.md

Create: `Skills/career-coaching-hotline/SKILL.md`

Follow the zo-hotline SKILL.md as a template. Include:
- Overview and purpose
- Setup instructions (env vars, VAPI config, Fillout form)
- Architecture description
- Feature list
- Security boundaries
- Phone number (TBD — V needs to provision in VAPI)
- Service details (port, URL, service ID)

### 4. VAPI Setup Guide

Create: `Skills/career-coaching-hotline/references/vapi-setup.md`

Step-by-step guide for V to:
1. Create a new assistant in VAPI dashboard
2. Set webhook URL to the deployed service
3. Configure the phone number
4. Set the Bearer Token credential for webhook auth
5. Test with a test call

### 5. Fillout Form Setup Guide

Create: `Skills/career-coaching-hotline/references/fillout-setup.md`

Guide for V to:
1. Create the form in Fillout.com (using form spec from D3.2)
2. Configure the webhook to POST to the intake endpoint
3. Test with a sample submission

### 6. Environment Variables Checklist

Document all required env vars and their status:
- `CAREER_HOTLINE_VOICE_ID` ✅ Set
- `VAPI_API_KEY` ✅ Set (shared with zo-hotline)
- `CAREER_HOTLINE_SECRET` ❓ V needs to create
- `CAREER_HOTLINE_PORT` (set by service registration)
- `CAREER_HOTLINE_CALENDLY_LINK` ❓ V needs to create

### 7. Git Commit

Stage and commit all new files to the feature branch:
```bash
git add Skills/career-coaching-hotline/ Knowledge/career-coaching-hotline/ Datasets/career-coaching-calls/ N5/config/PORT_REGISTRY.md
git commit -m "feat: Career Coaching Hotline - complete build"
```
