# D2.2: System Prompt Authoring (MANUAL — requires V review)

## Objective

Author the system prompt for the Career Coaching Hotline voice AI. This prompt defines the persona, voice, behavior, and domain of the hotline. It IS V — modeled on his actual personality and coaching style.

## Context

This is the most voice-sensitive Drop in the build. The system prompt is the soul of the hotline. It must sound like V giving career advice on the phone — not a generic chatbot, not a corporate HR advisor.

## Inputs

- V's psychographic profile: `N5/builds/career-coaching-hotline/artifacts/v-coaching-profile.md` (from D1.2)
- Extracted source material: `N5/builds/career-coaching-hotline/artifacts/extracted/` (from D1.1)
- Zoseph system prompt (structural reference): `Skills/zo-hotline/prompts/zoseph-system-prompt.md`
- V's voice lessons: `N5/prefs/voice/voice_lessons_v2.json`

## Structural Reference

Use Zoseph's system prompt as the structural template. The Career Coaching Hotline prompt should have equivalent sections but adapted for career coaching:

| Zoseph Section | Career Hotline Equivalent |
|----------------|--------------------------|
| Identity | V as career coach (not a character) |
| Voice Discipline | Same rules apply (1 question per turn, max 2 options, 2-3 sentences) |
| Opening | Career-focused opening (not Zo Computer focused) |
| Mode: Discover | Mode: Explore — caller is curious about career topics |
| Mode: Guide | Mode: Coach — caller has a specific career challenge |
| Take-Charge Behavior | Same — take the lead when caller is uncertain |
| Context | Career coaching context (Careerspan, V's background) |
| Character | V's actual personality (from psychographic profile) |
| What You Know About | Career topics organized by knowledge base categories |
| About the Creator | V IS the creator — different framing |
| Escalation | Careerspan booking — the natural next step |
| Feedback Collection | Same pattern |

## Deliverables

Create: `N5/builds/career-coaching-hotline/artifacts/career-coach-system-prompt.md`

### Required Sections

1. **Identity** — You are V's AI career coaching assistant, speaking in V's voice and style
2. **Voice Discipline** — Carry forward Zoseph's phone voice rules
3. **Opening** — Career-focused greeting
4. **Mode: Explore** — For callers who want general career guidance
5. **Mode: Coach** — For callers with specific challenges (resume help, interview prep, etc.)
6. **Mode: Pre-briefed** — For callers who submitted the Fillout form (phone number matched)
7. **Take-Charge Behavior** — Proactive diagnostic when caller is lost
8. **Career Diagnostic** — Socratic questions to understand career stage
9. **Value Prop Tree** — Personalized Careerspan sell based on caller's situation
10. **Character** — V's personality, communication style, beliefs
11. **Domain Knowledge** — What topics you can advise on
12. **Careerspan Positioning** — How to naturally introduce Careerspan as the deeper tool
13. **Privacy & What This Is** — Transparency about the hotline
14. **Escalation** — When and how to book a Careerspan session
15. **Feedback Collection** — Same pattern as Zoseph
16. **Tools** — Tool descriptions for the career diagnostic, recommendations, concept explanation, escalation, feedback
17. **Pre-briefed Caller Context** — How to use intake form data when phone number matches

## Critical Voice Requirements

- This sounds like V, not a corporate career counselor
- Direct and practical, not aspirational fluff
- Challenges conventional job search wisdom where V disagrees
- Grounds advice in V's decade of coaching experience
- Uses V's actual phrases and communication patterns (from voice profile)

## Success Criteria

- [ ] System prompt covers all required sections
- [ ] Voice discipline section is present and mirrors Zoseph quality
- [ ] Prompt reads as V, not generic
- [ ] Careerspan positioning is natural, not salesy
- [ ] Pre-briefed caller mode is well-defined
- [ ] All tools are described with proper names and parameters
