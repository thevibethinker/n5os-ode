# D4.1: Extract Voice Library Paragraph Chunks and Primitives

## Objective

Extract structured voice data from the existing voice library. This includes paragraph-level chunks and voice primitives that encode V's specific linguistic patterns.

## Data Sources

1. **Paragraph chunks** (150 entries): `/home/workspace/Knowledge/voice-library/paragraph-chunks.jsonl`
   - Keys: `id`, `text`, `post_type`, `position`, `char_count`, `source_post_id`
   - These are paragraph-level segments tagged by post type and position (opening/middle/closing)

2. **Voice primitives**: `/home/workspace/Knowledge/voice-library/voice-primitives.md`
   - Contains V's signature phrases, pivots, analogy frames, metaphor families
   - These are short but extremely high-signal for voice encoding

3. **LinkedIn primitives (ranked)**: `/home/workspace/N5/builds/voice-library-v2/linkedin_primitives_ranked.jsonl`
   - Pre-ranked voice primitives from previous analysis

4. **Corpus analysis report**: `/home/workspace/Knowledge/voice-library/corpus-analysis-report.md`
   - May contain additional examples or patterns

## Output

Write to: `N5/builds/lora-voice-evolution/artifacts/voice_library_raw.jsonl`

Each line:
```json
{
  "id": "vl_chunk_001",
  "source": "voice_library_chunks|voice_primitives|linkedin_primitives",
  "platform": "linkedin",
  "content_type": "paragraph_chunk|primitive|pattern",
  "text": "The text content",
  "char_count": 300,
  "timestamp": null,
  "metadata": {
    "post_type": "thought_leadership",
    "position": "opening",
    "primitive_type": "analogy_frame"
  }
}
```

## Requirements

1. For paragraph chunks: include all 150 entries (they're already curated).
2. For voice primitives: parse the markdown file and extract each primitive as a separate entry.
3. For ranked primitives: include entries with high ranking scores.
4. Tag each entry with its source for traceability.
5. Note: many paragraph chunks originate from LinkedIn posts that also appear in D1.1. Tag with `source_post_id` so D5.1 can decide how to handle overlaps (the chunk form may be more useful for training than the full post).
