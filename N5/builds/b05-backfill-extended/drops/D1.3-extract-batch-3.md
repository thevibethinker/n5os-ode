## Extraction Prompt

For each meeting, call the Zo API to extract items:

```python
import requests
import os
import json

def extract_items(content, meeting_id):
    prompt = f"""Extract Vrijen's action items from this meeting content.

For each item owned by Vrijen (V), return:
- title: Brief action title
- priority: external|strategic|urgent|normal  
- first_step: Specific 2-minute first action
- estimate_min: Time estimate in minutes
- domain: Careerspan|Zo|Personal|Other
- context: Brief context from meeting

Rules:
- ONLY items where V is the owner
- Skip vague items ("think about X", "consider Y")
- Skip items already marked done in the transcript
- Skip meeting logistics that aren't real tasks

Return JSON array. If no V-owned items, return empty array [].

MEETING: {meeting_id}

CONTENT:
{content}
"""
    
    response = requests.post(
        "https://api.zo.computer/zo/ask",
        headers={
            "authorization": os.environ["ZO_CLIENT_IDENTITY_TOKEN"],
            "content-type": "application/json"
        },
        json={"input": prompt},
        timeout=120
    )
    return response.json()["output"]
```
