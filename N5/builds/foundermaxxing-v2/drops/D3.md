# D3: Stripe Coupon + Promo Code System

## Objective

Build a coupon/promo code system for FounderMaxxing's Stripe products that allows V to:
1. Create named promo codes for whale accounts / strategic invites
2. Offer time-limited discounts for launch campaigns
3. Track which codes convert

## Requirements

### Promo Codes to Create
1. **FOUNDING** — For V's personal invites. Applies to Founding 15 product only.
2. **WHALE** — Custom discount for strategic accounts V wants to bring in (configurable %).
3. **LAUNCH2026** — Time-limited launch discount (e.g., first month free or % off first 3 months).

### Research Needed

### Implementation
- Create coupons via Stripe API or dashboard guidance
- Create promotion codes linked to those coupons
- Update payment links to allow promo codes at checkout
- Document the system so V can create new codes himself

- List active promo codes
- See redemption counts
