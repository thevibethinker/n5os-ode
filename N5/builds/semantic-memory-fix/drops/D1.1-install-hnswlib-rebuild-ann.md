# D1.1: Install hnswlib + Rebuild ANN Index

## Objective

Install the `hnswlib` Python package and rebuild the ANN (Approximate Nearest Neighbor) index for the semantic memory system. This is the single highest-impact fix — it will take search from 11-14 seconds (brute-force) to <100ms (indexed).

## Context

The semantic memory system in `N5/cognition/n5_memory_client.py` has an ANN fast path using hnswlib (HNSW algorithm), but `hnswlib` is not installed. Every search falls back to brute-force scan of all 190,593 vectors. The code for ANN search already exists and works — we just need the dependency and a rebuilt index.

### Key Files
- `N5/cognition/n5_memory_client.py` — Main memory client (lines 425-490: `_search_ann()` and `rebuild_ann_index()`)
- ANN index output: `N5/cognition/vectors_v2.hnsw` (index file) and `N5/cognition/vectors_v2.hnsw.ids` (block ID mapping)

### Database
- `N5/cognition/vectors_v2.db` — 190,593 vectors, all 3072-dim (OpenAI text-embedding-3-large)

## Tasks

### 1. Install hnswlib
```bash
pip install hnswlib
```
Verify with: `python3 -c "import hnswlib; print(hnswlib.__version__)"`

### 2. Verify ANN code path works
Before rebuilding, read the `rebuild_ann_index()` method (around line 446) in `n5_memory_client.py` and confirm it:
- Reads all vectors from the database
- Detects dimension (should be 3072)
- Builds HNSW index with cosine space
- Saves to `ANN_INDEX_PATH` (check what this constant is set to at the top of the file)
- Saves block ID mapping to `ANN_INDEX_PATH + ".ids"`

### 3. Rebuild the ANN index
```python
import sys
sys.path.insert(0, '/home/workspace')
from N5.cognition.n5_memory_client import N5MemoryClient

client = N5MemoryClient()
success = client.rebuild_ann_index(ef_construction=200, M=16)
print(f"Rebuild success: {success}")
```

**IMPORTANT**: This will take a few minutes for 190K vectors. The ef_construction=200 and M=16 are good defaults for recall/speed tradeoff.

### 4. Verify the index loads and searches work
```python
client2 = N5MemoryClient()  # Fresh instance
print(f"ANN index loaded: {client2.ann_index is not None}")
print(f"ANN block IDs count: {len(client2.ann_block_ids)}")

# Test search with ANN
import time
t0 = time.time()
results = client2.search("career coaching AI", limit=5)
t1 = time.time()
print(f"Search time: {t1-t0:.3f}s")
for r in results[:3]:
    print(f"  [{r.get('score', 0):.3f}] {r.get('path', '?')}")
```

**Success criteria**: Search time should be under 1 second (ideally <200ms for the vector part).

### 5. Check index freshness mechanism
```python
freshness = client2.check_index_freshness()
print(freshness)
```
