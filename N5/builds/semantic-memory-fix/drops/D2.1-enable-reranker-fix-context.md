# D2.1: Enable Reranker by Default + Fix n5_load_context.py

## Objective

Enable the FlashRank reranker by default in the semantic memory search pipeline and update `n5_load_context.py` to use it. The reranker dramatically improves result quality (tested: correct architectural principles files rank 1-3 vs irrelevant files without it), and adds <1s to search time.

## Context

### Current State
- `search()` in `n5_memory_client.py` has `use_reranker=False` as default
- `n5_load_context.py` calls `search_memory()` which calls `client.search()` without specifying `use_reranker`
- The FlashRank reranker is initialized, available, and working — just not turned on

### Quality Evidence (from audit)
Query: "P35 P36 architectural principles"
- **Without reranker**: Rank 2 = `Prompts/List View.prompt.md` (irrelevant)
- **With reranker**: Rank 1-3 = correct principles files

### Key Files
- `N5/cognition/n5_memory_client.py` — `search()` method (line ~618)
- `N5/scripts/n5_load_context.py` — `search_memory()` function (line ~56-85)
- `N5/cognition/rerankers.py` — FlashRank implementation (already working)

### Dependency
- **D1.1 must be complete** (ANN index available for fast initial retrieval)
- **D1.2 must be complete** (stale data purged so reranker doesn't waste time on dead results)

## Tasks

### 1. Change `search()` default in n5_memory_client.py

Change the `search()` method signature from:
```python
def search(self, query, ..., use_reranker=False, rerank_top_k=50, ...):
```
to:
```python
def search(self, query, ..., use_reranker=True, rerank_top_k=50, ...):
```

That's it for the memory client — one parameter default change.

### 2. Update `search_memory()` in n5_load_context.py

Read the `search_memory()` function in `N5/scripts/n5_load_context.py`. Currently it calls:
```python
client.search(query, limit=8)
```

Update it to explicitly pass reranker parameters:
```python
client.search(query, limit=8, use_reranker=True, rerank_top_k=30)
```

Using `rerank_top_k=30` instead of the default 50 since we only need 8 results — no need to rerank 50 candidates.

### 3. Verify the pipeline end-to-end

```bash
# Test n5_load_context.py with a semantic query
python3 N5/scripts/n5_load_context.py "P35 P36 architectural principles" 2>&1 | tail -20
```

Check that the memory_context block contains the correct principles files.

```bash
# Test with "build" mode
python3 N5/scripts/n5_load_context.py "build" 2>&1 | tail -20
```

### 4. Benchmark total latency

```python

import subprocess

                       capture_output=True, text=True, cwd='/home/workspace')

```

**Target**: Under 5 seconds total (was 14+ before).
