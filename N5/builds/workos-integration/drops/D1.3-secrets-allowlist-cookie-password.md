# Drop Brief: Configure secrets (cookie password + allowlist)

**Mission:** Add the remaining environment variables needed for secure WorkOS sessions and invite-only gating.

**Why manual:** Requires V to enter secrets in Zo settings.

**Output (artifacts):**
- Deposit only: `N5/builds/workos-integration/deposits/D1.3.json`

---

## Tasks

1) Generate a 32-character cookie password and store it as:
- `WORKOS_COOKIE_PASSWORD`

Suggested command:
```bash
openssl rand -base64 32
```

2) Add:
- `WORKOS_REDIRECT_URI` = `https://<PRODUCTION_PUBLIC_URL>/callback`
- `WHERES_V_ALLOWED_EMAILS` = comma-separated allowlist (case-insensitive)

---

## Success Criteria

- [ ] `WORKOS_COOKIE_PASSWORD` set in Zo secrets
- [ ] `WORKOS_REDIRECT_URI` set
- [ ] `WHERES_V_ALLOWED_EMAILS` set
