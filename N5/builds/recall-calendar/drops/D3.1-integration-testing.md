## Test Scenarios

### 1. Bot Creation Flow
```bash
# Create a test meeting in Google Calendar with Zoom link
# Wait for sync
python3 Integrations/recall_ai/calendar_scheduler.py sync --dry-run

# Verify bot would be scheduled
# Then run for real
python3 Integrations/recall_ai/calendar_scheduler.py sync
```

### 2. Webhook Flow
```bash
# Check webhook is receiving events
curl https://recall-webhook-va.zocomputer.io/health

# Check recent events
curl https://recall-webhook-va.zocomputer.io/events?limit=10
```

### 3. Meeting Deposit Flow

### 4. Edge Cases to Test
- [ ] Meeting with no video link (should skip)
- [ ] Meeting marked [NR] (should skip)
- [ ] Cancelled meeting (bot should be removed)
- [ ] Recurring meeting (each instance scheduled)
- [ ] Meeting time change (bot rescheduled)
- [ ] Bot fails to join (error handling)
- [ ] Recording permission denied (alert flow)
