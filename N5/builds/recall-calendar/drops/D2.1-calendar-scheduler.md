# D2.1: Calendar Scheduler Core

## Objective
Create the calendar sync logic that reads V's Google Calendar and auto-schedules Recall bots for meetings with video links.

## Context
- Zo has Google Calendar connected (use `use_app_google_calendar`)
- Recall bots should be scheduled 10+ minutes before meeting start
- Must detect: Zoom, Google Meet, Microsoft Teams links
- Must respect: meetings marked [NR] (No Record) or [SKIP]

## Architecture Decision
**Option A: Recall Calendar V2 (OAuth integration)**
- Recall manages calendar sync directly
- Requires additional OAuth setup in Recall dashboard
- More automated but less control

**Option B: Zo-managed scheduling (RECOMMENDED)**
- Zo polls calendar via existing Google Calendar integration
- Zo calls Recall API to schedule bots
- Full control over which meetings get recorded
- Can apply V's preferences/rules

Going with **Option B** for control and simplicity.

## Core Logic
```python
def sync_calendar_to_recall():
    # 1. Get upcoming calendar events (next 24-48 hours)
    events = get_upcoming_events()
    
    # 2. Filter to recordable meetings
    recordable = [e for e in events if should_record(e)]
    
    # 3. Check existing scheduled bots
    existing_bots = recall_client.list_bots(status="scheduled")
    
    # 4. Schedule missing bots
    for event in recordable:
        if not already_scheduled(event, existing_bots):
            schedule_bot(event)
    
    # 5. Cancel bots for deleted/changed meetings
    cleanup_orphaned_bots(events, existing_bots)
```

## Meeting Detection Rules
```python
def should_record(event):
    # Skip conditions
    if "[NR]" in event.title or "[SKIP]" in event.title:
        return False
    if event.status == "cancelled":
        return False
    if not has_video_link(event):
        return False
    
    # Record if has video link
    return True

def has_video_link(event):
    patterns = [
        r'zoom\.us/j/\d+',
        r'meet\.google\.com/[a-z-]+',
        r'teams\.microsoft\.com/.+',
    ]
    text = f"{event.description} {event.location} {event.conferenceData}"
    return any(re.search(p, text) for p in patterns)
```
