# D1.2 — Resume Intake Skill

## Objective

Create `Skills/careerspan-resume-intake/` — a skill that processes `[RESUME]` tagged emails and executes the candidate intake flow.

## Context

When Shivam forwards a resume with `[RESUME]` tag, this skill should:
1. Extract candidate info (name, email, resume)
2. Match to an existing Job Opening
3. Create a Candidate record in Airtable
4. Generate a Candidate Guide PDF
5. Upload to Drive
6. Email Shivam with summary and link

## Script Interface

```bash
python3 Skills/careerspan-resume-intake/scripts/process_resume.py \
  --email-subject "..." \
  --email-body "..." \
  --email-from "..." \
  --attachments "/path/to/resume.pdf" \
  [--job-opening-id "rec..."]  # Optional, if known
  [--dry-run]
```

## Job Matching Logic

1. Check email subject/body for company name or role
2. Query Airtable Job Openings for matches
3. Use LLM to confirm match if ambiguous
4. If no match found, flag for manual review
