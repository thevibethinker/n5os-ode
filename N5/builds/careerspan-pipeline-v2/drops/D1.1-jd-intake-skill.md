# D1.1 — JD Intake Skill

## Objective

Create `Skills/careerspan-jd-intake/` — a skill that processes `[JD]` tagged emails and executes the full JD intake flow.

## Context

When Shivam sends an email with `[JD]` in the subject, this skill should:
1. Extract the JD content
2. Create a Job Opening in Airtable
3. Generate a Hiring POV
4. Upload to shared Drive
5. Email Shivam with the POV link and any missing Core Questions
