# D1.2: Add Teaching Summary to Completion SMS

## Objective
Include pending teaching moments in the build completion SMS so V sees them immediately.

## Context
Currently `finalize_build()` sends this SMS:
```
[PULSE] {slug} FINALIZED ✅ Artifacts verified, tests passed.
```

It should also include any unacknowledged teaching moments from the build.

## Files to Modify
- `/home/workspace/Skills/pulse/scripts/pulse.py` — modify SMS generation in `finalize_build()`

## Implementation

After getting finalization results, fetch teaching summary and append to SMS:

```python
# 6. SMS summary (modify existing code)
teaching_summary = ""
try:
    teaching_cmd = subprocess.run(
        ["python3", str(WORKSPACE / "N5" / "pulse" / "teaching" / "build_review.py"),
         "generate", slug],
        capture_output=True, text=True, cwd=str(WORKSPACE)
    )
    if teaching_cmd.returncode == 0:
        teaching_data = json.loads(teaching_cmd.stdout)
        if teaching_data.get("total_moments", 0) > 0:
            terms = teaching_data.get("new_terms_introduced", [])
            teaching_summary = f"\n📚 Teaching: {', '.join(terms[:3])}"
            if len(terms) > 3:
                teaching_summary += f" +{len(terms)-3} more"
            teaching_summary += "\nReply 'teach' for details."
except:
    pass

if results["success"]:
    await send_sms(f"[PULSE] {slug} FINALIZED ✅ Artifacts verified, tests passed.{teaching_summary}")
else:
    # ... existing failure handling
```
