## Objective

Develop **R07 (Prediction)** — the block that extracts future state hypotheses, confidence levels, timeframes, and falsification criteria.

## R07 Domain: Predictive Thinking

### What R07 Sees
- Future state predictions
- Trend extrapolations
- Hypotheses about outcomes
- Timeframe estimates
- Confidence levels
- Falsification criteria
- Bet-worthy beliefs

### What R07 Ignores
- Current market state → R04
- Strategic choices → R03
- Content about predictions → R09

## Analysis Dimensions for R07

1. **Prediction Type:**
   - Market/industry
   - Technology
   - Personal/career
   - Careerspan-specific

2. **Timeframe:**
   - Near-term (< 6 months)
   - Medium-term (6-24 months)
   - Long-term (> 2 years)

3. **Confidence:**
   - High (would bet on it)
   - Medium (plausible)
   - Low (speculation)

4. **Falsifiability:**
   - Clear criteria
   - Partial criteria
   - Hard to test

5. **Stakes:**
   - High (major decisions depend on this)
   - Medium (influences planning)
   - Low (intellectual interest)

## Output Schema

```markdown
## R07: Prediction

**Prediction:** [One-line statement]
**Timeframe:** [Near | Medium | Long]
**Confidence:** [High | Medium | Low]

### The Prediction
[2-3 paragraphs developing the prediction]

### Evidence/Reasoning
> [Quote and supporting reasoning]

### Falsification Criteria
[How would we know this is wrong?]

### Check-in Date
[When to revisit this prediction]

### Memory Links
- Related predictions: [...]
- Supporting observations: [...]
```

## Special Feature: Tracking Hook

R07 should include a mechanism to revisit predictions. Output should suggest a calendar reminder or tracking entry.
