## Objective

Develop **R00 (Emergent)** — the special block for content that doesn't fit R01-R09, with a path to promote to new block types.

## R00 Domain: Everything Else

### What R00 Sees
- Valuable content that doesn't fit other blocks
- Potential new categories
- Edge cases between blocks
- Novel insight types

### R00's Special Role
- Catch-all for meaningful content
- Incubator for new block types
- Quality gate (not everything is worth capturing)

## Analysis Dimensions for R00

1. **Why Not Other Blocks:**
   - [Explicit reasoning for each considered block]

2. **Provisional Category:**
   - Suggested name for this type of insight
   - Similar to: [nearest block]

3. **Recurrence Check:**
   - First time seeing this type
   - Seen before (link to prior R00s)
   - Third occurrence (promotion trigger)

4. **Value Assessment:**
   - High (definitely capture)
   - Medium (capture but lower priority)
   - Low (maybe not worth it)

5. **Promotion Criteria:**
   - What would make this a new R-block?
   - How many more instances needed?

## Special Feature: Promotion Tracking

R00 output should check for prior R00s with similar provisional labels. If 3+ exist, flag for potential new block type creation.
