# D2.1: FounderMaxxing Landing Page on zo.space

## Objective

Build a high-quality landing page at va.zo.space/foundermaxxing with:
- Value proposition and positioning (non-technical founders first)
- What they'll build (reference N5/OS catalog of real systems)
- Session structure (3-phase: show & tell → build-along → variance)
- Pricing tiers (Founding Class $100/mo, Class of 2026 $200/mo)
- Zo Mentorship standalone ($150/mo)
- Built-in application form (name, LinkedIn, company website, goals, desired use cases)
- Payment link integration from W1 deposits
- Mobile responsive
- Distinctive design — use frontend-design skill for high quality

## Key Messaging

- One-liner: "Fuck falling behind." (use selectively in design)
- Clean tagline: "Weekly AI builds for non-technical founders who refuse to get left behind."
- V's positioning: "I'm the best non-technical coder most people have met."
- Non-technical advantage: qualitative thinking is a strength, not a weakness
- Pedagogy: meta OS heuristics, not tool tutorials
- Social proof: Next Play session (80+ signups), real systems built

## Form Fields

1. Full name (required)
2. LinkedIn URL (required)
3. Company website (required)
4. What are you hoping to achieve? (textarea, required)
5. What AI use cases would you most like to explore? (textarea, optional)

## Form Backend

Create an API route at /api/foundermaxxing-apply that:
1. Validates form data
2. Stores submission to a JSON file in workspace
3. Returns success confirmation

## Dependencies

Needs payment link URLs from D1.1, D1.2, D1.3 deposits.

## Design Reference

Read frontend-design skill for design principles. This should look premium — matching the $100-200/mo price point. Dark theme preferred (matches V's brand).
