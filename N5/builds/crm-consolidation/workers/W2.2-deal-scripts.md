## Key Schema Changes

```sql
-- OLD: deal_contacts was standalone
SELECT * FROM deal_contacts WHERE associated_deal_id = ?

-- NEW: Join through deal_roles
SELECT p.*, dr.role, dr.context
FROM people p
JOIN deal_roles dr ON p.id = dr.person_id
WHERE dr.deal_id = ?

-- OLD: Add contact
INSERT INTO deal_contacts (full_name, company, ...) VALUES (?, ?, ...)

-- NEW: Two-step process
-- 1. Create or find person
INSERT OR IGNORE INTO people (full_name, email, company) VALUES (?, ?, ?)
-- 2. Create role linking them to deal
INSERT INTO deal_roles (deal_id, person_id, role) VALUES (?, ?, ?)
```
