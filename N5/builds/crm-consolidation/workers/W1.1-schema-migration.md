# Worker Brief: W1.1 - Schema & Migration

## Objective

Create the unified n5_core.db database schema and migration script to consolidate data from three source databases.

## Scope

### Files to CREATE
- `N5/scripts/n5_core_schema.py` — Schema definition and creation
- `N5/scripts/migrate_to_n5_core.py` — Data migration from all sources

### Files to READ (do not modify)
- `N5/data/deals.db` — Source: deals, deal_contacts, deal_activities
- `N5/data/crm_v3.db` — Source: profiles, organizations, calendar_events
- `Personal/Knowledge/CRM/db/crm.db` — Source: individuals, organizations, relationships

### Must NOT Touch
- Any existing script in N5/scripts/ (that's Wave 2+)
- Path constants (that's W1.2)
- Markdown profile files

## MUST DO

1. **Create `n5_core_schema.py`:**
   - Full schema from PLAN.md Phase 1.1
   - CLI with `--create`, `--reset`, `--verify` flags
   - Enable foreign keys: `PRAGMA foreign_keys = ON`
   - Create all indexes

2. **Create `migrate_to_n5_core.py`:**
   - Backup existing DBs before migration
   - Migrate in order: organizations → people → deals → deal_roles → rest
   - Deduplicate people by email (case-insensitive)
   - Log all migration actions
   - Handle conflicts gracefully (log, don't crash)
   - CLI with `--dry-run`, `--source [all|crm|crm_v3|deals]`, `--verify`

3. **Run migration and verify:**
   - Execute full migration
   - Verify record counts
   - Verify foreign key integrity
   - Output summary report

## MUST NOT DO

- Modify any existing scripts
- Delete source databases (mark deprecated only)
- Change path constants (W1.2 handles this)

## Expected Output

1. `N5/scripts/n5_core_schema.py` exists and runs without errors
2. `N5/scripts/migrate_to_n5_core.py` exists and runs without errors
3. `N5/data/n5_core.db` created with all tables
4. Migration summary showing:
   - People migrated: ~250+ (after dedupe)
   - Organizations: ~30+
   - Deals: 99
   - Deal roles: 33+
   - Calendar events: count from crm_v3

## Verification Commands

```bash
# Schema creation
python3 N5/scripts/n5_core_schema.py --create --verify

# Migration dry run
python3 N5/scripts/migrate_to_n5_core.py --dry-run

# Full migration
python3 N5/scripts/migrate_to_n5_core.py --source all

# Verify
sqlite3 N5/data/n5_core.db "SELECT 'people', COUNT(*) FROM people UNION SELECT 'deals', COUNT(*) FROM deals UNION SELECT 'organizations', COUNT(*) FROM organizations"
```
