## Backup Strategy

### 1. Complete Backup Location
```bash
# Create timestamped backup
BACKUP_DIR="/home/workspace/Backups/Meetings/pre-v3-migration-$(date +%Y%m%d-%H%M%S)"
```

### 2. Backup Commands
```bash
# Create backup directory
mkdir -p "$BACKUP_DIR"

# Full copy of Personal/Meetings/
cp -r /home/workspace/Personal/Meetings/ "$BACKUP_DIR/Personal-Meetings-full/"

# Compressed archive for space efficiency  
cd /home/workspace
tar -czf "$BACKUP_DIR/personal-meetings-backup.tar.gz" Personal/Meetings/

# Verification
echo "Backup verification:"
echo "Original size: $(du -sh Personal/Meetings/)"
echo "Backup size: $(du -sh "$BACKUP_DIR")"
echo "File count original: $(find Personal/Meetings/ -type f | wc -l)"
echo "File count backup: $(find "$BACKUP_DIR/Personal-Meetings-full/" -type f | wc -l)"
```
