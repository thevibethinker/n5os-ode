# D4.2: Block Generator (MANUAL DROP)

## Objective
Build block generation orchestrator that spawns /zo/ask per block.

## Why Manual
Block generation is the most token-expensive part. V wants to:
- Review prompt quality in real-time
- Tune generation parameters
- Catch quality issues early

## Tasks
1. Create `Skills/meeting-ingestion/scripts/block_generator.py`
2. For each selected block:
   - Load prompt from Prompts/Blocks/
   - Call /zo/ask with transcript + prompt
   - Save output to meeting folder
   - Log to manifest
3. Handle failures gracefully
4. Support retry for specific blocks

## Acceptance Criteria
- [ ] Generator works for all block types
- [ ] Failures handled gracefully
- [ ] Manifest updated with generation status
- [ ] Retry mechanism works
