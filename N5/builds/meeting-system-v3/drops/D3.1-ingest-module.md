# D3.1: Build Ingest Module

## Objective
Build the ingest module that normalizes transcripts and creates v3 manifests.

## Tasks
1. Create `Skills/meeting-ingestion/scripts/ingest.py`
2. Handle formats: .md, .txt, .docx, .jsonl
3. Convert jsonl to markdown (speaker: text format)
4. Generate microsummary via /zo/ask
5. Create v3 manifest with status="ingested"
6. CLI: `python3 ingest.py <file_or_folder> [--dry-run]`
