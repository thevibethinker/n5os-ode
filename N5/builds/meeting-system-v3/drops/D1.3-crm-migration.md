# D1.3: CRM Migration

## Objective
Migrate CRM database from legacy location to canonical location, add is_internal flag, and seed internal team members.

## Context
- Current location: `Personal/Knowledge/Legacy_Inbox/crm/crm.db`
- Target location: `N5/data/crm.db`
- Need to add `is_internal` column to individuals table
- Seed internal team: V, Logan, Ilse, Shivam, Ankith

## Tasks

### 1. Backup Original
```bash
cp Personal/Knowledge/Legacy_Inbox/crm/crm.db N5/data/crm_backup_$(date +%Y%m%d).db
```

### 2. Copy to Canonical Location
```bash
cp Personal/Knowledge/Legacy_Inbox/crm/crm.db N5/data/crm.db
```

### 3. Add is_internal Column
```sql
ALTER TABLE individuals ADD COLUMN is_internal BOOLEAN DEFAULT FALSE;
```

### 4. Seed Internal Team
```sql
-- V is always internal
UPDATE individuals SET is_internal = TRUE WHERE name LIKE '%Vrijen%' OR name LIKE '%Attawar%';

-- Core team
UPDATE individuals SET is_internal = TRUE WHERE name LIKE '%Logan%';
UPDATE individuals SET is_internal = TRUE WHERE name LIKE '%Ilse%';
UPDATE individuals SET is_internal = TRUE WHERE name LIKE '%Shivam%';
UPDATE individuals SET is_internal = TRUE WHERE name LIKE '%Ankith%';
```

### 5. Verify
```sql
SELECT name, is_internal FROM individuals WHERE is_internal = TRUE;
```

## Output
Write deposit with migration results and list of internal team members flagged.

- [ ] CRM copied to N5/data/crm.db
