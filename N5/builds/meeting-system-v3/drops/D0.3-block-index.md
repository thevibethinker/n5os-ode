# D0.3: Create BLOCK_INDEX.yaml

## Objective
Create a canonical block index with metadata, recipes for different meeting scenarios, and trigger conditions.

## Context
- V wants "a block index in the block folder that has particular formulas or combinations for specific types of meetings or specific scenarios"
- This makes the system more dynamic — recipes can be selected based on meeting type, participants, or explicit triggers
- The smart selector (D0.4) will use this index

## Input Files
- `file 'N5/builds/meeting-system-v3/PLAN.md'` — Final block list
- `file 'N5/builds/meeting-system-v3/deposits/D0.1.json'` — Audit findings
- `file 'Personal/config/priorities.yaml'` — V's priorities

## Tasks

### 1. Create Block Metadata
For each block, document:
- Code and name
- Purpose (one sentence)
- When to generate (always/conditional/trigger)
- Trigger conditions (if applicable)
- Average output length
- Priority relevance (which of V's priorities it serves)

### 2. Define Recipes
Create preset combinations for scenarios:

```yaml
recipes:
  external_standard:
    description: "Default for external meetings"
    always: [B00, B01, B02, B03, B08, B21, B26]
    conditional: [B04, B06, B07, B10, B13, B14, B25, B28, B32, B33]
    
  external_sales:
    description: "Sales/partnership meetings"
    always: [B00, B01, B02, B03, B06, B08, B21, B26, B28]
    conditional: [B04, B07, B10, B13, B14, B25, B32, B33]
    
  external_investor:
    description: "Investor meetings"
    always: [B00, B01, B02, B03, B06, B08, B21, B26, B28, B33]
    conditional: [B04, B10, B13, B25, B32]
    
  internal_standup:
    description: "Team standups"
    always: [B00, B40, B41, B42, B47]
    conditional: [B43, B44, B45, B46, B48]
    
  internal_strategy:
    description: "Strategy/planning sessions"
    always: [B00, B40, B41, B42, B45, B47, B48]
    conditional: [B43, B44, B46]
```

### 3. Define Triggers
Map verbal cues to block activation:

```yaml
triggers:
  B07_warm_intro:
    patterns: ["intro me to", "introduce me to", "connect me with"]
    via: B00
    
  B14_blurb:
    patterns: ["draft a blurb", "write a blurb", "blurb about"]
    via: B00
```

### 4. Priority Mapping
Link blocks to V's priorities:

```yaml
priority_relevance:
  business:  # Careerspan recruiting revenue
    high: [B02, B03, B06, B08, B25, B28]
    medium: [B01, B13, B21, B33]
```
