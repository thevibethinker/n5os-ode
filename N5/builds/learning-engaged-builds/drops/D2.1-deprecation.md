## Context

The old teaching system used regex-based pattern detection to identify "imprecise language" from V and surface technical terms. This was a lightweight v1 approach (built in the pulse-v2 build, Jan 2026). We're replacing it entirely with LLM-native learning mode where the orchestrator handles teaching natively.

Three Python scripts are being removed: `moment_generator.py`, `teaching_manager.py`, `build_review.py`. These are called by three live callers: `review_manager.py`, `feedback_extractor.py`, and `sms_intake.py`. All calls must be removed cleanly — no dangling imports, no broken code paths.

The Teacher persona must be updated to read from `N5/config/understanding_bank.json` instead of its inline Technical Calibration section.
