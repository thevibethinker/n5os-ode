### Changes to pulse.py

1. **Find `process_tidying_complete()` function** (around line 770-800)

2. **After the aggregation call, add fix dispatcher:**

The function likely has something like:
```python
# Aggregate findings
aggregation_result = subprocess.run([
    "python3", str(WORKSPACE / "N5" / "pulse" / "tidying" / "aggregator.py"),
    "aggregate", slug
], capture_output=True, text=True)
```

Add AFTER this:

```python
    # NEW: Dispatch auto-fixes for high-confidence findings
    config = load_config()
    auto_fix_threshold = config.get("auto_fix", {}).get("confidence_threshold", 0.9)
    
    if config.get("auto_fix", {}).get("enabled", True):
        print(f"  Dispatching auto-fixes (threshold: {auto_fix_threshold})...")
        dispatch_result = subprocess.run([
            "python3", str(WORKSPACE / "N5" / "pulse" / "tidying" / "fix_dispatcher.py"),
            "dispatch", slug,
            "--threshold", str(auto_fix_threshold)
        ], capture_output=True, text=True)
        
        if dispatch_result.returncode == 0:
            try:
                dispatch_data = json.loads(dispatch_result.stdout)
                fixes_dispatched = dispatch_data.get("fixes_dispatched", 0)
                if fixes_dispatched > 0:
                    print(f"  ✓ Auto-fixed {fixes_dispatched} issues")
                escalated = dispatch_data.get("escalated", 0)
                if escalated > 0:
                    print(f"  ⚠ Escalated {escalated} issues for manual review")
            except json.JSONDecodeError:
                print(f"  Fix dispatcher output: {dispatch_result.stdout}")
        else:
            print(f"  Fix dispatcher warning: {dispatch_result.stderr or 'unknown error'}")
```
