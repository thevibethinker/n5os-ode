Clarify which meeting-related builds are superseded by the `meeting-ingestion` skill and which are still active/needed.

## Context

- The `meeting-ingestion` skill at `Skills/meeting-ingestion/` is the current active system
- It unified MG-1 through MG-6 agents into a single skill
- Need to determine which builds are historical artifacts vs still needed

## For Each Build

1. Read PLAN.md or STATUS.md
2. Determine relationship to current `meeting-ingestion` skill
3. Classify as:
   - `superseded` - replaced by meeting-ingestion skill
   - `historical` - one-time build, no longer relevant
   - `active` - still being used
   - `reference` - contains patterns/docs worth keeping
