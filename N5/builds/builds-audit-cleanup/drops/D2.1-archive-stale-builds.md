# D2.1: Archive Stale Builds

## Objective

Move all builds classified as stale/superseded/historical to `N5/builds/_archive/` with a manifest.

## Pre-requisites

- D1.1, D1.2, D1.3 must be complete
- Read their deposits to get the full list of builds to archive

## Process

1. Read deposits from D1.1, D1.2, D1.3
2. Compile list of builds to archive (status: complete, abandoned, superseded, historical)
3. Create `N5/builds/_archive/` if not exists
4. Move each build folder to archive
5. Create `N5/builds/_archive/MANIFEST.md` with:
   - List of archived builds
   - Original status and archive reason
   - Date archived

## Safety

- Check for `.n5protected` before moving - skip if protected
- Do NOT archive builds with status `active` or `blocked`
- Preserve git history (just move, don't delete)

## Output

Deposit JSON:
```json
{
  "builds_archived": <count>,
  "builds_skipped": <count>,
  "skipped_reasons": ["<reason1>"],
  "manifest_path": "N5/builds/_archive/MANIFEST.md",
  "space_reclaimed": "<size>"
}
```
