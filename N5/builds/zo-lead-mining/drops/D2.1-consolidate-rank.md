# D2.1: Consolidate + Rank Leads

## Objective

Read all 4 Wave 1 deposits, merge them into a single deduplicated ranked lead list, and produce the final artifact.

## Instructions

1. Read all deposit files:
   - `/home/workspace/N5/builds/zo-lead-mining/deposits/D1.1.json`
   - `/home/workspace/N5/builds/zo-lead-mining/deposits/D1.2.json`
   - `/home/workspace/N5/builds/zo-lead-mining/deposits/D1.3.json`
   - `/home/workspace/N5/builds/zo-lead-mining/deposits/D1.4.json`
2. Merge all leads into a single list
3. Deduplicate by name (fuzzy match — "Alex Wu" and "Alex" at same org = same person)
4. When a person appears in multiple deposits:
   - Take the HIGHEST score across deposits
   - Merge archetypes (union of all)
   - Combine rationales
   - Note all sources
5. Sort by score descending, then alphabetically within same score
6. Produce final ranked list as both JSON deposit and markdown artifact

- **Given** D1.1 has "Alex Wu" score 3 and D1.3 has "Alex Wu" score 4

- **Then** output single entry with score 4 and combined rationale

- **Verify** total_from_wave1 correctly sums all deposits including zero-lead ones
