# D1.2: Merge GitHub Content into Local Export

**Root Cause Identified:** A force push to GitHub (commit `773efa8`, Jan 26 21:46 UTC) from Devin replaced the entire 36-commit history with a single mega-commit. The push included new Skills/ content but obliterated git history.

**Goal:** Preserve local's 36-commit history while adding GitHub's new content.

**Current State:**
- GitHub: Has Skills/meeting-ingestion/, Skills/pulse/, N5/scripts/meeting_*.py, and other new files
- Local: Has 36 commits of history but missing the new content

## Strategy

```bash
# 1. Clone fresh from GitHub
cd /tmp && rm -rf n5os-ode-github-fresh
git clone https://github.com/vrijenattawar/n5os-ode.git n5os-ode-github-fresh

# 2. Copy new content
cd /home/workspace/N5/export/n5os-ode
cp -r /tmp/n5os-ode-github-fresh/Skills/meeting-ingestion Skills/
cp -r /tmp/n5os-ode-github-fresh/Skills/pulse Skills/
cp /tmp/n5os-ode-github-fresh/N5/scripts/meeting_*.py N5/scripts/

mkdir -p templates/configs
cp /tmp/n5os-ode-github-fresh/templates/configs/drive_locations.yaml.template templates/configs/

# 3. Copy any new B-blocks
for f in /tmp/n5os-ode-github-fresh/Prompts/Blocks/Generate_B*.prompt.md; do
  base=$(basename "$f")
  if [ ! -f "Prompts/Blocks/$base" ]; then
    cp "$f" Prompts/Blocks/
  fi
done

# 4. Handle install.sh (compare and merge)
# Local has uncommitted changes - inspect diff and decide

# 5. Commit
git add -A
git commit -m "feat: Merge GitHub content (Skills/, meeting scripts) while preserving history"
```
