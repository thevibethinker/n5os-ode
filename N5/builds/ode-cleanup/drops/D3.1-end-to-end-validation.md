# D3.1: End-to-End Validation

## Context

All cleanup and protection work is complete. This Drop performs final validation to ensure:
1. GitHub repo matches local export
2. Fresh clone works correctly
3. install.sh functions properly
4. No broken file references

## Scope

### MUST DO
1. Push protection mechanisms from D2.1 to GitHub
2. Clone fresh from GitHub to `/tmp/n5os-ode-validation`
3. Run `validate_repo.py` on fresh clone
4. Verify file counts match between local and remote
5. Test install.sh in simulated workspace
6. Check for broken file references in prompts
7. Generate final comparison report
8. Clean up the D1.1 backup (no longer needed after validation passes)

### MUST NOT DO
- Do NOT make content changes
- Do NOT force push (regular push only)

### Files You Own
- PUSH: GitHub repo (regular push for D2.1 changes)
- DELETE: `/home/workspace/N5/export/n5os-ode-backup-pre-merge/` (only after validation passes)
- CREATE: `N5/builds/ode-cleanup/artifacts/validation-report.md`

## Strategy

```bash
# 1. Push D2.1 changes to GitHub
cd /home/workspace/N5/export/n5os-ode
git add -A
git commit -m "feat: Add history protection workflow and sync protocol" || echo "Nothing to commit"
git push origin main

# 2. Clone fresh
cd /tmp && rm -rf n5os-ode-validation
git clone https://github.com/vrijenattawar/n5os-ode.git n5os-ode-validation

# 3. Compare file counts
echo "Local export files: $(find /home/workspace/N5/export/n5os-ode -type f -not -path '*/.git/*' -not -name '*.pyc' | wc -l)"
echo "Fresh clone files:  $(find /tmp/n5os-ode-validation -type f -not -path '*/.git/*' | wc -l)"

# 4. Run validate_repo.py if it exists
if [ -f /tmp/n5os-ode-validation/scripts/validate_repo.py ]; then
    cd /tmp/n5os-ode-validation && python3 scripts/validate_repo.py
fi

# 5. Test install.sh simulation (dry run)
mkdir -p /tmp/install-test-workspace
cd /tmp/install-test-workspace
cp -r /tmp/n5os-ode-validation/* .
# Simulate install by checking what it would do
head -50 install.sh

# 6. Check for broken file references in prompts
echo "Checking for broken file references..."
for prompt in Prompts/*.prompt.md Prompts/Blocks/*.prompt.md; do
    if [ -f "$prompt" ]; then
        # Extract file references and check they exist
        grep -oE "file '[^']+'" "$prompt" | while read ref; do
            path=$(echo "$ref" | sed "s/file '//;s/'//")
            if [[ "$path" != /* ]] && [ ! -f "$path" ]; then
                echo "WARN: $prompt references non-existent: $path"
            fi
        done
    fi
done

# 7. Compare commit counts
echo "Local commits:  $(cd /home/workspace/N5/export/n5os-ode && git log --oneline | wc -l)"
echo "GitHub commits: $(cd /tmp/n5os-ode-validation && git log --oneline | wc -l)"

# 8. If all passes, clean up backup
rm -rf /home/workspace/N5/export/n5os-ode-backup-pre-merge
```
