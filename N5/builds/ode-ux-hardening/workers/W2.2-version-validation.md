# N5OS Ode Troubleshooting Guide

Common issues and how to fix them.

## Installation Issues

### Files ended up in n5os-ode/ subfolder instead of root

**Problem:** After running install, all files are in `/home/workspace/n5os-ode/` instead of `/home/workspace/`.

**Solution:**
```bash
cd /home/workspace
cp -r n5os-ode/* .
rm -rf n5os-ode/
```

### "Permission denied" when running install.sh

**Problem:** Can't execute the install script.

**Solution:**
```bash
chmod +x install.sh
bash install.sh
```

### Git clone fails (network/firewall)

**Problem:** Can't clone the repository.

**Solution:** Download manually:
1. Go to https://github.com/vrijenattawar/n5os-ode
2. Click "Code" → "Download ZIP"
3. Extract to your workspace
4. Run `bash install.sh`

## Bootloader Issues

### "Persona already exists" but it doesn't appear in list

**Problem:** Bootloader says persona exists, but you can't find it.

**Solution:** There may be a persona with a similar name. Check Settings → Your AI → Personas for any "Ode" or "Operator" personas.

### Bootloader fails partway through

**Problem:** Bootloader encountered an error and stopped.

**Solution:** 
1. It's safe to run again — it skips existing items
2. Run `@BOOTLOADER.prompt.md` again
3. If it keeps failing at the same step, report the error

### Rules not applying

**Problem:** Created rules but they don't seem to work.

**Solution:** Rules apply to NEW conversations. Start a fresh conversation to see them in effect.

## Database Issues

### "Database not found" errors

**Problem:** Scripts fail because databases don't exist.

**Solution:** Initialize them:
```bash
# Conversations DB
python3 N5/scripts/conversation_sync.py init

# Semantic Memory DB
python3 -c "
import sqlite3
import os
os.makedirs('N5/cognition', exist_ok=True)
conn = sqlite3.connect('N5/cognition/brain.db')
conn.execute('CREATE TABLE IF NOT EXISTS resources (id TEXT PRIMARY KEY)')
conn.commit()
conn.close()
print('✅ Created brain.db')
"
```

### Semantic memory not finding anything

**Problem:** Searches return no results.

**Solution:** You need to index your files first:
```python
from N5.cognition.n5_memory_client import N5MemoryClient
client = N5MemoryClient()
client.index_file("/path/to/your/file.md")
```

## Rollback

### Need to completely start over

**Problem:** Installation is broken and you want to reset.

**Solution:**
```bash
# If you have a backup
python3 N5/scripts/rollback.py

# If no backup, manual cleanup
rm -rf N5/ Knowledge/ Records/ Prompts/Blocks/
rm -f BOOTLOADER.prompt.md PERSONALIZE.prompt.md SETUP_PROGRESS.md
# Then reinstall fresh
```
