## MUST DO

### 1. Create N5/scripts/backup.py

```python
#!/usr/bin/env python3
"""
N5OS Ode Backup - Create snapshot before install/upgrade
Usage: python3 backup.py [--list | --create]
"""

Features:
- Creates timestamped backup directory: .n5os-backup-{YYYYMMDD-HHMMSS}/
- Backs up: N5/, Prompts/, Knowledge/, Records/, *.prompt.md at root
- Creates manifest.json listing what was backed up
- --list: Shows available backups
- --create: Creates new backup
- Excludes: node_modules/, .git/, __pycache__/, *.db (too large)
- Prints: "✅ Backup created: .n5os-backup-20260119-143000/"
```

### 2. Create N5/scripts/rollback.py

```python
#!/usr/bin/env python3
"""
N5OS Ode Rollback - Restore from backup
Usage: python3 rollback.py [backup_name]
"""

Features:
- Without args: Lists available backups, prompts for selection
- With arg: Restores specified backup
- Reads manifest.json to know what to restore
- Removes N5OS files added since backup
- Restores backed up files
- Prints: "✅ Rolled back to .n5os-backup-20260119-143000/"
```

### 3. Update BOOTLOADER.prompt.md

**Major changes:**

A. **Slim down by reading personas from files:**
   Replace the inline persona definitions with:
   ```
   ### 1.1 Create Ode Operator
   
   1. Read the persona definition from `N5/personas/operator.md`
   2. Extract the YAML frontmatter for the name
   ```

   For personas:
   ```
   Before creating each persona:
   - List existing personas
   - If name matches: SKIP with message
   - If no match: CREATE with message
   ```
   
   For rules:
   ```
   Before creating each rule:
   - List existing rules  
   - Check if instruction text matches (fuzzy match OK)
   - If similar exists: SKIP with message
   - If no match: CREATE with message
   ```
   
   For folders:
   ```
   mkdir -p is already idempotent, but add:
   - If exists: "⏭ Folder N5/ already exists"
   - If created: "✅ Created N5/"
   ```
   
   For files:
   ```
   Before creating prefs.md, context_manifest.yaml, etc:
   - If exists AND not placeholder: "⏭ Skipping [file] (already configured)"
   - If exists AND is placeholder: "🔄 Updating [file]"
   - If not exists: "✅ Created [file]"
   ```
