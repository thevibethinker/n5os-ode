   - A Zo Computer account
   - Terminal access to your workspace

   ## Order of Operations

   ### Step 1: Quick Install (Terminal)
   Run this single command in your terminal:
   ```bash

   ```
   @BOOTLOADER.prompt.md
   ```

   ### Step 3: Personalize (Zo Conversation)
   In the same or new conversation:
   ```
   @PERSONALIZE.prompt.md
   ```
   ✅ This configures your name, timezone, and preferences

   ### Step 4: Verify Installation (Optional)
   To confirm everything worked:
   ```
   @VALIDATE.prompt.md
   ```
   ✅ This runs a health check and reports any issues

   ## Check Your Progress
   
   Open `SETUP_PROGRESS.md` in your workspace to see which steps are complete.

   ## Troubleshooting

   **Files ended up in a subfolder?**
   The install script should move everything to workspace root. If it didn't, manually move the contents of `n5os-ode/` to your workspace root.

   **Bootloader failed partway through?**
   It's safe to run again — it skips anything already created.

   **Need to start over?**
   Run `python3 N5/scripts/rollback.py` to restore your pre-install state.
   ```
