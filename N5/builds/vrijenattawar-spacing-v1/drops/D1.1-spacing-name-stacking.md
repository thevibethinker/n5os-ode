# D1.1: Spacing & Name Stacking

## Context

You are editing `Sites/vrijenattawar-staging/src/pages/PersonalLanding.tsx` — a personal landing page for Vrijen Attawar at vrijenattawar.com.

**Problem being solved:**
1. The name "Vrijen Attawar" has an optical centering issue — the space between words sits at mathematical center, but "Attawar" is visually heavier, making the composition feel off-balance
2. Vertical spacing is uneven — too much gap between roles and buttons, content sits too high
3. Logo is fixed-size and too large on mobile

## Scope

**Files you own:**
- `Sites/vrijenattawar-staging/src/pages/PersonalLanding.tsx`

**Responsibilities:**
- Stack the name on two lines (Vrijen above, Attawar below) for optical centering

- Make logo responsive

**Must NOT touch:**
- Button layout structure (Stream 2 handles this)
- Footer structure (Stream 2 handles this)
- Easter egg logic (Konami code, badge clicks) — preserve exactly as-is

## MUST DO

1. **Stack the name** — Change the `<h1>` from single-line to stacked:
   ```jsx
   <h1 className="mb-6 flex flex-col items-center text-5xl font-bold tracking-normal sm:text-6xl lg:text-7xl antialiased">
     <span>
       <span className="text-zinc-500">V</span>
       <span className="text-white">rijen</span>
     </span>
     <span className="text-zinc-500">Attawar</span>
   </h1>
   ```
