# D2.1: Interactive Review & Import

## Context

D1.2 has staged all extracted action items. Now V needs to:
1. See all items in a single batch checklist
2. Tick items to approve
3. Bulk import approved items
4. Discard unticked items

**This is a manual Drop** — run in an interactive thread with V.

## Scope

### Responsibilities
- Generate checklist from staged items
- Present to V
- Parse V's selections
- Promote approved items
- Dismiss rejected items

### Must Not Touch
- Meeting files
- Prompt files
- Task database directly (use stage.py promote/dismiss)

## MUST DO

### Step 1: List Staged Items
```bash
python3 Skills/task-system/scripts/stage.py list --json
```

### Step 2: Generate Checklist

Format for V:
```markdown
## Task Backfill Review

**Instructions:** Check the boxes for tasks you want to import. Leave unchecked to discard.

### Careerspan Tasks

- [ ] **Follow up with Sarah about partnership** | external | 5min
  - First step: Open email to Sarah, write 2-sentence check-in
  - Source: 2026-01-20 Anna Bao meeting
  - Urgency: ⚠️ External commitment

- [ ] **Review investor memo** | strategic | 15min
  - First step: Open memo draft, read first section
  - Source: 2026-01-21 David sync
  - Urgency: None

### Zo Tasks

- [ ] **Fix calendar integration bug** | urgent | 30min
  ...

### Personal Tasks

- [ ] **Schedule dentist appointment** | normal | 5min
  ...
```

### Step 3: V Reviews

Present checklist to V. Wait for V to return with checked items.

### Step 4: Process Selections

For each checked item:
```bash
python3 Skills/task-system/scripts/stage.py promote --id STAGE_ID
```

For each unchecked item:
```bash
python3 Skills/task-system/scripts/stage.py dismiss --id STAGE_ID
```

### Step 5: Confirm Import

```bash
python3 Skills/task-system/scripts/task.py list
```

Show V the imported tasks.

## Expected Output

After V completes review:
```json
{
  "status": "complete",
  "items_presented": 18,
  "items_approved": 12,
  "items_dismissed": 6,
  "tasks_now_in_system": 12,
  "notes_for_orchestrator": "V reviewed all items. 12 promoted, 6 dismissed."
}
```
