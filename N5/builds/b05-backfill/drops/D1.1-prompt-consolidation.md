# D1.1: B05 Prompt Consolidation

## Context

Two B05 prompts exist:
- `Prompts/Blocks/Generate_B05.prompt.md` — Old table-based format (v2.0)
- `Prompts/Generate_B05_Action_Items.prompt.md` — New decomposition format (v1.0, created 2026-01-25)

The new prompt has task decomposition metadata (priority, first_step, estimate, domain, project, milestones) that the task system needs.

## Scope

### Files
- `Prompts/Blocks/Generate_B05.prompt.md` — REPLACE contents
- `Prompts/Generate_B05_Action_Items.prompt.md` — DELETE after merge

### Responsibilities
- Merge new prompt content into block system location
- Ensure YAML frontmatter matches block conventions
- Clean up redundant file

### Must Not Touch
- Any other prompt files
- Meeting files
- Task system scripts

## MUST DO

1. Read current content of both prompt files
2. Take the body/instructions from `Generate_B05_Action_Items.prompt.md` 
3. Merge into `Prompts/Blocks/Generate_B05.prompt.md` with proper frontmatter:
   ```yaml
   ---
   description: Generate B05 Action Items block with structured decomposition
   tags: [meeting-intelligence, block-generation, b05, action-items, task-decomposition]
   tool: true
   created: 2025-11-03
   last_edited: 2026-01-26
   version: 3.0
   provenance: <your_conversation_id>
   ---
   ```
4. Delete `Prompts/Generate_B05_Action_Items.prompt.md`
5. Verify the merged file exists and has decomposition keywords

## MUST NOT DO

- Change the extraction rules/logic of the prompt
- Create new directories
- Touch any meeting files
