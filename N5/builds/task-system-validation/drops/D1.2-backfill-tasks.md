## Phase 2: Structure Tasks

For each task discovered, structure it properly:

```python
task = {
    "title": "Clear, actionable title (2-6 words)",
    "description": "What needs to be done and why",
    "domain": "careerspan|zo|personal",
    "project": "Optional project name",
    "priority_bucket": "strategic|external|urgent|normal",
    "due_at": "YYYY-MM-DD if known",
    "estimated_minutes": 30,  # Be generous
    "first_step": "The 2-5 minute action to start"
}
```

### Priority Guidelines
- **external**: Someone else is waiting (investor, client, partner)
- **urgent**: Has a deadline within 48 hours
- **strategic**: Advances long-term goals (thought leadership, systems, growth)
- **normal**: Everything else

## Phase 3: Create Tasks

Use the task.py CLI to create each task:

```bash
python3 Skills/task-system/scripts/task.py add "Task title" \
  --domain careerspan \
  --priority external \
  --due 2026-01-28 \
  --estimate 30
```

Or use db.py directly for bulk creation:

```python
from Skills.task-system.scripts.db import create_task, get_or_create_domain

domain_id = get_or_create_domain("careerspan")
task_id = create_task(
    title="Task title",
    domain_id=domain_id,
    priority_bucket="external",
    due_at="2026-01-28",
    estimated_minutes=30,
    description="What needs to be done"
)
```
