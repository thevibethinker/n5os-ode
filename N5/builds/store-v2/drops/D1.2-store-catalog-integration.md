# D1.2: Store Catalog Integration

## Context

The store at store-va.zocomputer.io currently has hardcoded product data in the React components. This drop creates a `catalog.json` data file and updates the store to read from it, enabling the export pipeline (D1.1) to add products by updating the catalog.

## Scope

### Files You Own
- `Sites/store-va/data/catalog.json` (create)
- `Sites/store-va/src/pages/StoreLanding.tsx` (modify - read from catalog)
- `Sites/store-va/src/pages/ProductDetail.tsx` (modify - read from catalog)
- `Sites/store-va/src/lib/catalog.ts` (create - catalog loader)

### Must NOT Touch
- `Skills/*` (D1.1 owns the export pipeline)
- `Sites/store-va/src/components/*` (no component changes needed)
- `Sites/store-va/server.ts` (Phase 4 scope)
