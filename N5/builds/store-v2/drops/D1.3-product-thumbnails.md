# D1.3: Product Thumbnail Generation

## Context

Building a Gumroad-style store at https://store-va.zocomputer.io selling Zo skills. Need consistent, visually striking thumbnails for product cards.

## Products Needing Thumbnails

1. **Pulse: Build Orchestration** — Automated parallel build system that spawns workers, monitors health, validates outputs
2. **Dynamic Survey Analyzer** — Transforms Fillout form responses into insights with statistical analysis and dashboards

## Visual Direction

- **Consistent motif** across all thumbnails (same style family)
- **Dark aesthetic** to match site (black/zinc background, sky-400 accents)
- **Abstract/symbolic** — not literal screenshots
- **Square format** (1:1 aspect ratio, 512x512 or 1024x1024)

## Your Task

1. **Ideate** with V on symbolic concepts for each product
2. **Generate** thumbnails using image generation tools
3. **Iterate** based on V's feedback until approved
4. **Save** final images to:
   - `/home/workspace/Sites/store-va/public/images/pulse-thumb.png`
   - `/home/workspace/Sites/store-va/public/images/survey-analyzer-thumb.png`

## Suggested Concepts (Starting Points)

**Pulse:**
- Parallel streams/rivers converging
- Neural network nodes pulsing with energy
- Water drops falling into orchestrated ripples
- Circuit board with flowing data paths

**Survey Analyzer:**
- Data points crystallizing into patterns
- Abstract bar charts transforming into insights
- Funnel visualization with light emerging
- Geometric shapes revealing hidden structure

## MUST DO
- Generate at least 2-3 options per product for V to choose
- Maintain consistent visual language across both
- Save final approved images to the paths above

## MUST NOT DO
- Use literal screenshots or UI mockups
- Create busy/cluttered compositions
- Use bright/white backgrounds (must be dark)
