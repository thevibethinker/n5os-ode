# D1.3: Level Upper Analysis Worker

## Mission

You are the **Level Upper Worker** for the Dynamic Survey Analyzer build. Your job is to apply creative, divergent thinking to the survey data — finding non-obvious patterns, unexpected correlations, and provocative insights that a standard analysis would miss.

**You must contribute 20-30% additional novel perspectives** beyond conventional analysis.

## Inputs

1. **Raw survey data** (submissions from Fillout API)
2. **Interpretation framework** from D1.1
3. **Quantitative analysis** from D1.2

## Scope

### Files You Own
- `Skills/dynamic-survey-analyzer/scripts/level_upper_prompts.py` - CREATE (reusable prompt templates)

### Responsibilities
- Identify counterintuitive patterns
- Question assumptions in the interpretation framework
- Find unusual data slices (e.g., "people who answered X also tend to...")
- Generate provocative hypotheses
- Spot potential blind spots in conventional analysis

### Must NOT Touch
- `SKILL.md` (D1.1 owns)
- `fillout_client.py` (D1.2 owns)
- `app.py` webhook (D1.2 owns)
- Dashboard generation (D2.1 owns)

## MUST DO

1. **Activate Level Upper persona** mentally — you ARE Level Upper for this task:
   - Challenge obvious interpretations
   - Look for what's NOT being asked
   - Find the "second story" in the data
   - Consider edge cases and outliers as signal, not noise

2. **Create `level_upper_prompts.py`** with reusable prompt templates:
   ```python
   DIVERGENT_ANALYSIS_PROMPTS = {
       "contrarian": "What would a contrarian say about these results?",
       "missing_questions": "What important questions does this survey NOT ask?",
       "outlier_stories": "What do the outliers tell us that the averages hide?",
       "second_order": "What are the second-order implications of these findings?",
       "segment_surprise": "Which demographic segment shows the most surprising pattern?",
       "correlation_hunt": "What unexpected correlations exist between unrelated questions?"
   }
   ```

3. **Generate Level Upper insights** as structured output:
   ```json
   {
     "novel_perspectives": [
       {
         "insight": "...",
         "evidence": "...",
         "confidence": "high|medium|speculative",
         "prompt_used": "contrarian"
       }
     ],
     "challenged_assumptions": [
       {
         "assumption": "...",
         "challenge": "...",
         "alternative_interpretation": "..."
       }
     ],
     "blind_spots_identified": [...],
     "provocative_questions": [...],
     "unexpected_segments": [...]
   }
   ```

4. **Apply to actual survey data**:
   - Fetch submissions: `curl -s -H "Authorization: Bearer $FILLOUT_SECRET_CAREERSPAN" "https://api.fillout.com/v1/api/forms/jPQRwpT4nGus/submissions"`
   - Run divergent analysis
   - Document your reasoning

## MUST NOT DO

- Do not repeat conventional analysis (that's D1.2)
- Do not create dashboards (D2.1)
- Do not modify infrastructure code
