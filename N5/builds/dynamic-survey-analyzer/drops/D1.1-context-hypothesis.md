# D1.1: Context & Hypothesis Worker

## Mission

You are the **Context & Hypothesis Worker** for the Dynamic Survey Analyzer build. Your job is to ingest survey structure and create an interpretation framework that guides all subsequent analysis.

## Inputs

1. **Survey metadata** from Fillout API:
   - Form name and description
   - Question list with types (multiple choice, text, scale, etc.)
   - Answer options for structured questions

2. **First submissions** (if available):
   - Sample responses to understand data shape

## Scope

### Files You Own
- `Skills/dynamic-survey-analyzer/SKILL.md` - CREATE the skill manifest

### Responsibilities
- Create interpretation framework document
- Generate hypotheses about expected patterns
- Identify which questions are high-signal vs. demographic
- Map question relationships (e.g., "Q3 likely correlates with Q7")

### Must NOT Touch
- `fillout_client.py` (D1.2 owns this)
- `app.py` webhook (D1.2 owns this)
- Dashboard generation (D2.1 owns this)

## MUST DO

1. **Read the Fillout form structure** for the target survey:
   ```bash
   curl -s -H "Authorization: Bearer $FILLOUT_SECRET_CAREERSPAN" \
     "https://api.fillout.com/v1/api/forms/jPQRwpT4nGus" | jq '.questions'
   ```

2. **Create SKILL.md** at `Skills/dynamic-survey-analyzer/SKILL.md`:
   - Frontmatter with name: `dynamic-survey-analyzer`
   - Description explaining the automated survey analysis system
   - Usage instructions (manual trigger + auto-trigger via webhook)
   - List of scripts and their purposes

3. **Create interpretation framework** as a JSON structure:
   ```json
   {
     "form_id": "...",
     "form_name": "...",
     "analysis_hypotheses": [...],
     "question_classification": {
       "high_signal": [...],
       "demographic": [...],
       "open_ended": [...]
     },
     "expected_correlations": [...],
     "themes_to_watch": [...]
   }
   ```

4. **Write framework to deposit** for D1.2 and D1.3 to consume

## MUST NOT DO

- Do not write analysis scripts (D1.2's job)
- Do not generate visualizations (D2.1's job)
- Do not modify the webhook (D1.2's job)

## Context

This is part of the **Dynamic Survey Analyzer** build. The system will:
1. Auto-trigger when first submission arrives for a new Fillout form
2. Run 4 workers: Context (you), Analysis (D1.2), Level Upper (D1.3), Synthesis (D2.1)
3. Generate an ongoing analysis file + Plotly dashboard
4. Create a 30-day scheduled agent for daily updates

Your framework directly guides D1.2 (what to measure) and D1.3 (what to challenge).
