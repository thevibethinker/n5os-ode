# W1.3: Loop Runner Script

## Context

This is the core Ralph concept: run an AI agent in a loop until specifications are met. Each iteration starts with **fresh context** (not accumulated). Progress persists in files and git, not in the agent's memory.

The loop:
```
while not done:
    run agent with prompt
    check backpressure (tests, lint)
    check for struggle
    if all pass: done
    else: iterate
```

## MECE Declaration

**SCOPE:** Create loop runner script and prompt template

**MUST DO:**
1. Create `N5/scripts/loop_runner.py` with CLI interface
2. Core loop logic:
   - Read prompt from file (Markdown)
   - Call /zo/ask API with fresh context each iteration
   - After each iteration: run backpressure validation
   - Check struggle detector
   - Stop conditions: backpressure passes, max iterations, stuck detected
3. Safety features:
   - Require `--execute` flag to actually run (default is dry-run)
   - Max 20 iterations before requiring `--force` or human confirmation
   - Log each iteration to `loop_history.jsonl` in build folder
4. Create template at `N5/templates/loop_prompt.md`:
   ```markdown

   ## Specification
   {specification}
   
   ## Current State

   {file_list}
   
   ## Success Criteria
   {criteria}
   
   ## Instructions
   1. Assess current state
   2. Make incremental progress toward specification
   3. Run tests/validation before declaring done
   4. Write completion status to {output_file}
   ```
5. Include `--help` with clear usage
6. Support `--dry-run` to show execution plan without running

- Exceed iteration limits without explicit override

## /zo/ask API Reference

```python
import requests
import os

response = requests.post(
    "https://api.zo.computer/zo/ask",
    headers={
        "authorization": os.environ["ZO_CLIENT_IDENTITY_TOKEN"],
        "content-type": "application/json"
    },
    json={"input": prompt_text}
)
result = response.json()
output = result["output"]
```

## Example Usage

```bash
# Dry run (default) - shows plan
python3 N5/scripts/loop_runner.py --prompt task.md --build-slug my-build

# Actually execute
python3 N5/scripts/loop_runner.py --prompt task.md --build-slug my-build --execute

# With iteration limit
python3 N5/scripts/loop_runner.py --prompt task.md --max-iterations 5 --execute

# Force past 20 iterations (dangerous)
python3 N5/scripts/loop_runner.py --prompt task.md --max-iterations 50 --force --execute
```
