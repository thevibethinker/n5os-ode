## V-OS Tag Reference

The `[SKIP]` tag is part of V's V-OS tagging system. It appears in calendar event descriptions like:
```
V-OS Tags: {Zo} [SKIP] *
```
or simply:
```
[SKIP]
```

Any event with `[SKIP]` anywhere in its description should be excluded from meeting metrics.
