## Sparkline Reference

Characters for sparkline (8 levels):
```
▁ ▂ ▃ ▄ ▅ ▆ ▇ █
```

Map values to levels:
```python
SPARK_CHARS = "▁▂▃▄▅▆▇█"
def sparkline(values):
    if not values:
        return ""
    min_v, max_v = min(values), max(values)
    if max_v == min_v:
        return SPARK_CHARS[4] * len(values)
    return "".join(
        SPARK_CHARS[int((v - min_v) / (max_v - min_v) * 7)]
        for v in values
    )
```
