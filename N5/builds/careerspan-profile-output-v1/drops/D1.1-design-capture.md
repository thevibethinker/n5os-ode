# D1.1: Design Capture

## Objective

Extract visual design tokens and aesthetic guidelines from mycareerspan.com to inform the final output.

## Instructions

1. Navigate to https://www.mycareerspan.com
2. Capture the following design elements:
   - **Color palette**: Primary, secondary, accent colors (hex values)
   - **Typography**: Font families, sizes, weights for headings and body
   - **Spacing**: Margins, padding patterns
   - **Visual style**: Rounded vs sharp corners, shadows, gradients
   - **Tone**: Professional, modern, approachable, etc.
3. Note any recurring UI patterns (cards, sections, CTAs)
4. Provide Gamma-compatible guidance (Gamma uses its own styling system, so translate to general terms)

## Output

Write to: `N5/builds/careerspan-profile-output-v1/drops/D1.1-design-spec.md`

Include:
- Color palette with hex codes
- Typography recommendations
- Visual style summary (2-3 sentences)
- Gamma styling hints
