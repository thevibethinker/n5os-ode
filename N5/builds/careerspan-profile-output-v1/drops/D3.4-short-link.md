# D3.4: Create Short Link

## Objective

Create a short.io link pointing to the Gamma page for easy sharing.

## Prerequisites

Read: `N5/builds/careerspan-profile-output-v1/output/gamma-url.txt`

## Instructions

1. Get the Gamma URL from the output file
2. Use the short.io API to create a shortened link
3. If short.io API key is available in environment (`SHORTIO_API_KEY`), use it
4. If not available, note in issues and output the raw Gamma URL as fallback

### Short.io API

```bash
curl -X POST "https://api.short.io/links" \
  -H "Authorization: $SHORTIO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "domain": "link.short.io",
    "originalURL": "<gamma_url>"
  }'
```

Or use the Careerspan domain if configured.

## Output

Save to: `N5/builds/careerspan-profile-output-v1/output/short-link.txt`

Format:
```
Short URL: <short.io link>
Original URL: <gamma URL>
Created: <timestamp>
```
