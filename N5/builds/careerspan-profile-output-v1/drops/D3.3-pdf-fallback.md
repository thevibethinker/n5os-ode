# D3.3: Generate PDF Fallback

## Objective

Generate a PDF version of the candidate profile as a fallback if Gamma fails or for offline sharing.

## Prerequisites

Read: `N5/builds/careerspan-profile-output-v1/drops/D3.1-assembled-content.md`

## Instructions

1. Use pandoc to convert the assembled markdown to PDF:
   ```bash
   pandoc D3.1-assembled-content.md -o H-profile.pdf \
     --pdf-engine=xelatex \
     -V geometry:margin=1in \
     -V fontsize=11pt
   ```

2. If xelatex isn't available, use wkhtmltopdf or similar

3. Ensure the PDF is:
   - Readable
   - Properly formatted (headings, tables intact)
   - Single document (not split across files)

## Output

Save to: `N5/builds/careerspan-profile-output-v1/artifacts/H-profile.pdf`
