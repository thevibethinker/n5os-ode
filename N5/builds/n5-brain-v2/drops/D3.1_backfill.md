   - Store entities and relationships via graph_store

   - Check relationships table exists and has data
   - Report entity/relationship counts

4. Integration testing:
   - Test `search("networking")` returns both vector AND graph results

   - Benchmark: compare search quality before/after graph

5. Document results:

   - Top entity types (how many PERSON, CONCEPT, etc.)
