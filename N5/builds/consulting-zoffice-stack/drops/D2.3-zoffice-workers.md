# Drop 2.3: Zoffice Workers Framework

## Objective
Create the persona activation system for zoputer. Workers are scheduled personas that activate during specific conditions (time-based or event-based).

## Requirements

### Worker Manifest System

Each worker has a manifest in `ZOFFICE_WORKERS/`:

```yaml
# ZOFFICE_WORKERS/librarian.manifest
name: Librarian
persona_id: 1bb66f53-9e2a-4152-9b18-75c2ee2c25a3
type: scheduled
schedule: "0 9 * * *"  # 9 AM ET daily
timezone: America/New_York
activation:
  command: "python3 N5/scripts/librarian_worker.py"
notifications:
  on_start: false
  on_complete: true
  on_error: true
  method: email
```

### Worker Types

**1. Librarian (Scheduled)**
- Schedule: Daily, 9 AM ET
- Persona: Librarian
- Task: Run export pipeline (Drop 3.1)
- Activation: Automatic

**2. Consultant (Event-Based)**
- Trigger: Calendly booking webhook
- Persona: Teacher (consulting mode)
- Task: Activate 15 min before session, assist during, summarize after
- Activation: Manual (requires V confirmation for first session)

**3. Debugger (Scheduled)**
- Schedule: Weekly, Sunday 10 AM ET
- Persona: Debugger
- Task: Health check, audit verification, error triage
- Activation: Automatic

### Worker Runner

Build `N5/scripts/zoffice_worker.py`:

```bash
# List all workers
python3 N5/scripts/zoffice_worker.py list

# Activate specific worker
python3 N5/scripts/zoffice_worker.py activate librarian

# Check worker status
python3 N5/scripts/zoffice_worker.py status

# Deactivate all
python3 N5/scripts/zoffice_worker.py deactivate-all
```

### Activation Flow

1. Worker scheduled/triggered
2. Set persona via `set_active_persona()`
3. Load context via `python3 N5/scripts/n5_load_context.py <context>`
4. Execute worker command
5. Log completion to audit system
6. Return to default persona

### Calendly Integration for Consultant

- Webhook from Calendly → zoputer endpoint
- Extract: client_email, meeting_time, event_type
- Activate Consultant persona 15 min before meeting_time
- Deactivate 30 min after meeting_time (or on manual signal)
