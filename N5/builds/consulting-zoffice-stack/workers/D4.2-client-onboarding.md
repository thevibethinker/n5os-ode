# Drop 4.2: Client Onboarding Flow

## Objective
Create the complete onboarding experience for new consulting clients—setting expectations, gathering context, and establishing the advisory relationship.

## Deliverables

### 1. Welcome Email Template

Subject: "Your Zo setup journey starts now — here's what to expect"

Content sections:
- Welcome + context (they just booked/talked to V)
- What zoputer is (advisory Zo, not V's personal Zo)
- Communication model (email-based, office hours)
- Security note (all comms logged, no secrets shared)
- First step: complete intake questionnaire
- Calendly link for first working session

### 2. Intake Questionnaire

Build as Google Form or Airtable form:

**Section 1: Context**
- What prompted you to explore Zo?
- What have you tried so far?
- What's your biggest friction point right now?

**Section 2: Environment**
- Current tools (Notion, Gmail, Calendar, etc.)?
- Integrations you use daily?
- Team size and technical sophistication?

**Section 3: Goals**
- What does "success" look like in 30 days?
- What does "success" look like in 90 days?
- Any hard constraints (time, budget, tools)?

**Section 4: Preferences**
- Preferred communication style (brief/detailed)?
- Best time for async updates?
- Anything else I should know?

### 3. First 30 Days Roadmap

Template document shared after intake:

```
Week 1: Discovery & Foundation
- Review intake responses
- Initial setup call (Calendly)
- Identify 2-3 quick wins
- Begin environment audit

Week 2: Quick Wins
- Implement 1-2 high-impact automations
- Document existing workflows
- Set up basic folder structure

Week 3: Core Build
- Deploy primary skill(s) for use case
- Test and refine
- Client training on usage

Week 4: Review & Plan
- Measure against 30-day goals
- Gather feedback
- Define 60-90 day roadmap
```

### 4. Onboarding Checklist (for zoputer)

Internal tracking:
```markdown
## Client: {{name}} | Onboarded: {{date}}

- [ ] Welcome email sent
- [ ] Intake questionnaire completed
- [ ] Initial call scheduled
- [ ] First 30 days roadmap shared
- [ ] Environment audit complete
- [ ] Quick wins identified
- [ ] Primary skill deployed
- [ ] 30-day check-in scheduled
```
