# Drop 2.1: Archetype Zo Account Setup

## Objective
Set up the zoputer.zo.computer account as the "Archetype Zo"—a clean mirror of V's Level 1 content, ready to receive exports and serve clients.

## Manual Steps Required (V must complete)

### 1. Account Creation
- [ ] Sign up at zo.computer with handle `zoputer`
- [ ] Confirm email verification
- [ ] Complete onboarding wizard

### 2. API Key Setup
- [ ] Navigate to [Settings > Developers]
- [ ] Create API key named `ZO_CONSULTING_API`
