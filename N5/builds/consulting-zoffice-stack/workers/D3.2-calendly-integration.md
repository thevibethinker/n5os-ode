# Drop 3.2: Calendly Integration

## Objective
Connect Calendly booking events to Consultant worker activation on zoputer. When a client books time, zoputer prepares for the session automatically.

## Requirements

### Calendly Setup (V must complete)

1. [ ] Go to Calendly → Integrations → Webhooks
2. [ ] Add webhook URL: `https://zoputer.zo.computer/api/calendly/webhook`
3. [ ] Subscribe to events:
   - `invitee.created` (booking made)
   - `invitee.canceled` (booking canceled)
4. [ ] Copy webhook signing secret to zoputer environment

### Webhook Handler (zoputer-side)

Build route on zoputer at `/api/calendly/webhook`:

```typescript
// Accepts Calendly webhook payload
// Validates signature
// Parses event type
// Triggers appropriate action
```

Actions:
- `invitee.created` → Schedule Consultant activation
- `invitee.canceled` → Cancel scheduled activation

### Session Preparation Flow

When booking received:
1. Parse: client_email, client_name, event_time, event_type
2. Create session record in `CONSULTING_MANIFEST.md`
3. Schedule Consultant worker activation for (event_time - 15 min)
4. Send prep email to V with client context

Prep email to V:
```
Subject: [Zoffice] Consulting session scheduled - {{client_name}}

Session Details:
- Client: {{client_name}} ({{client_email}})
- Time: {{event_time}} ET
- Type: {{event_type}}
- Your Calendly link was used

Zoputer will activate Consultant persona at {{activation_time}} ET.
You'll receive a briefing 5 minutes before the session.

No action needed unless you want to review the client's prior emails.
```

### Consultant Activation

15 minutes before session:
1. Activate Consultant persona on zoputer
2. Load client context (prior emails, session history)
3. Send briefing to V via SMS:
   "[Zoffice] Session with {{client_name}} in 15 min. Consultant active. Reply 'ready' to confirm."

During session:
- Consultant monitors zoputer email for client messages
- Can draft responses for V to review/send
- Logs all activity to audit system

After session:
- Auto-deactivate Consultant (30 min after scheduled end)
- Generate session summary
- Update CONSULTING_MANIFEST.md

### Manual Override

V can text "zoffice hold" to pause auto-activation
V can text "zoffice resume" to re-enable
