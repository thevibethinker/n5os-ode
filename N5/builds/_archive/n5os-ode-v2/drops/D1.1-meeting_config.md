# D1.1: Create Meeting Config Module

## Context
We're adding Meeting Ingestion to the public n5os-ode repo. All meeting scripts currently have hardcoded `/home/workspace` paths. This Drop creates the central config module that all other scripts will import from.

## Scope
```yaml
files:
  - N5/scripts/meeting_config.py (CREATE)
must_not_touch:
  - Any other meeting scripts
  - Skills/meeting-ingestion/
```

## MUST DO
1. Create `N5/scripts/meeting_config.py` with environment variable support:
   - ZO_WORKSPACE (default: /home/workspace)
   - MEETINGS_DIR (default: WORKSPACE/Personal/Meetings)
   - STAGING_DIR (default: MEETINGS_DIR/Inbox)
   - LOG_DIR (default: WORKSPACE/N5/logs)
   - REGISTRY_DB (default: WORKSPACE/N5/data/meeting_registry.db)
   - TIMEZONE (default: UTC)
   - ENABLE_SMS (default: false)
2. Include docstring explaining each variable
3. Test: `python3 -c "from N5.scripts.meeting_config import *; print(WORKSPACE)"`

## MUST NOT DO
- Modify any other files
- Add complex logic

## EXPECTED OUTPUT
