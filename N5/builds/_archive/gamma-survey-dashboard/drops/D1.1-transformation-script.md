# D1.1: Create Transformation Script

## Objective
Create a Python script that reads the survey analysis markdown and generates a Gamma webpage via the API.

## Context
- Source: `Datasets/survey-analyses/jPQRwpT4nGus/analysis.md`
- Gamma skill: `Skills/gamma/scripts/gamma.ts`
- Output: Gamma webpage URL

## Requirements

1. **Read analysis.md** and extract:
   - Response count (N=X)
   - Last updated timestamp
   - Executive summary
   - All data tables (Tool Adoption, Challenges, Usage Level, Value Drivers, Sentiment)
   - Changes from previous count
   - Recommendations
   - Methodology

2. **Format for Gamma** as structured inputText:
   - Title with response count and timestamp
   - Executive summary as key findings
   - Data tables preserved exactly
   - Visual-friendly layout hints

3. **Call Gamma API** via the skill:
   ```bash
   bun run Skills/gamma/scripts/gamma.ts generate "<inputText>" \
     --format webpage \
     --text-mode preserve \
     --amount detailed \
     --images pictographic \
     --theme blue-steel \
     --wait
   ```

4. **Return** the generated URL

## Output Location
- Script: `N5/scripts/gamma_survey_generator.py`
- Should be callable as: `python3 N5/scripts/gamma_survey_generator.py`
- Prints the Gamma URL on success

## Acceptance Criteria
- [ ] Script reads analysis.md correctly
- [ ] Extracts all key data tables
- [ ] Generates valid Gamma inputText
- [ ] Calls Gamma API successfully
- [ ] Returns URL
