# D2.1: Create Scheduled Agent

## Objective
Set up a scheduled agent that regenerates the Gamma dashboard daily at 8am ET, running through Jan 30.

## Agent Specification

**Schedule**: Daily at 8:00 AM ET (13:00 UTC)
**End Date**: Jan 30, 2026
**Delivery**: Email

**RRULE**: `FREQ=DAILY;BYHOUR=13;BYMINUTE=0;UNTIL=20260131T000000Z`

**Instruction**:
```
Gamma Survey Dashboard Regeneration

1. Run the regeneration script:
   python3 N5/scripts/gamma_survey_generator.py

2. Capture the new Gamma URL from output.

3. Update URL history:
   - Read N5/builds/gamma-survey-dashboard/artifacts/url_history.json
   - Append new entry with url, timestamp, response_count
   - Save file

4. Email V with:
   Subject: [Survey Dashboard] Updated - N responses
   Body:
   - New dashboard URL (clickable)
   - Response count change since last update
   - Brief summary of any significant metric changes
   - Note: Dashboard will auto-update daily through Jan 30

5. If generation fails, email V with error details.
```

## Tasks

1. Create the scheduled agent using `create_agent` tool
2. Verify agent is created and scheduled correctly
3. Document the agent ID for future reference

- [ ] Agent created with correct RRULE
