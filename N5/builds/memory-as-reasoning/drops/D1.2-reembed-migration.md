# D1.2: Re-Embedding Migration Pipeline

## Context
Migrate all 48K+ vectors from `brain.db` (384-dim, all-MiniLM-L6-v2) to `brain_v2.db` (3072-dim, text-embedding-3-large).

**This is the most expensive phase (~$3 OpenAI cost, ~60-90 min runtime).**

## Your Scope (MECE)
You own ONLY:
- `N5/cognition/brain_v2.db` (CREATE)
- `N5/scripts/migrate_embeddings.py` (CREATE)
- `N5/cognition/n5_memory_client.py` — ONLY the DB selector logic (UPDATE)

You do NOT touch:
- `brain.db` (read-only source)
- `reasoning.db` (that's D2.1)
- Feature flags file (that's D1.1)

## Tasks

### 1. Create Migration Script
Create `N5/scripts/migrate_embeddings.py` with:

**Requirements:**
- Read all blocks from `brain.db`
- Embed each block's content via OpenAI `text-embedding-3-large`
- Write to `brain_v2.db` with proper schema
- **Checkpoint every 500 blocks** (save progress to `N5/cognition/migration_checkpoint.json`)
- **Resume capability** (check for checkpoint on start)
- **Rate limiting** (respect OpenAI limits, add backoff)
- **Progress logging** (print every 100 blocks)

**Schema for brain_v2.db:**
```sql
CREATE TABLE metadata (
    key TEXT PRIMARY KEY,
    value TEXT
);

CREATE TABLE resources (
    id TEXT PRIMARY KEY,
    path TEXT NOT NULL UNIQUE,
    hash TEXT,
    last_indexed_at DATETIME,
    content_date DATETIME
);

CREATE TABLE blocks (
    id TEXT PRIMARY KEY,
    resource_id TEXT NOT NULL,
    block_type TEXT,
    content TEXT NOT NULL,
    start_line INTEGER,
    end_line INTEGER,
    token_count INTEGER,
    content_date DATETIME,
    FOREIGN KEY(resource_id) REFERENCES resources(id) ON DELETE CASCADE
);

CREATE TABLE vectors (
    block_id TEXT PRIMARY KEY,
    embedding BLOB NOT NULL,
    embedding_dim INTEGER DEFAULT 3072,
    FOREIGN KEY(block_id) REFERENCES blocks(id) ON DELETE CASCADE
);
```

**Insert into metadata:**
```sql
INSERT INTO metadata VALUES ('embedding_model', 'text-embedding-3-large');
INSERT INTO metadata VALUES ('embedding_dim', '3072');
INSERT INTO metadata VALUES ('migrated_from', 'brain.db');
INSERT INTO metadata VALUES ('migration_date', '<ISO timestamp>');
```

### 2. Add DB Selector to n5_memory_client.py
Update `N5/cognition/n5_memory_client.py` to support selecting which DB to use:

```python
# In __init__, add:
self.embedding_store = self._load_feature_flag("N5_EMBEDDING_STORE", "legacy")
if self.embedding_store == "v2":
    self.db_path = str(BRAIN_DB).replace("brain.db", "brain_v2.db")
    # Also update embedding provider to match v2
    self.provider = "openai"
    self.openai_model = "text-embedding-3-large"
```

Add helper method:
```python
def _load_feature_flag(self, key: str, default: Any) -> Any:
    flag_path = Path("/home/workspace/N5/config/feature_flags.json")
    if flag_path.exists():
        flags = json.loads(flag_path.read_text())
        return flags.get(key, default)
    return default
```
