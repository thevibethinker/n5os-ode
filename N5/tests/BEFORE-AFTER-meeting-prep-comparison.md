## User Experience

### Before (Cold Start)
```
1. Read calendar
2. Search emails (limited to 3 messages)
3. Guess at context
4. Generate generic prep
5. Hope for the best in meeting
6. Forget context after meeting
```
**Time:** 15-20 min per meeting  
**Quality:** Low confidence, generic

---

### After (Profile-Based)
```
1. Load profile (instant)
2. Review relationship context
3. Check recent updates
4. Generate specific prep
5. Enter meeting confident
6. Profile auto-updates post-meeting
```
**Time:** 5-10 min per meeting  
**Quality:** High confidence, specific
