# Interview Loop (Draft)

Purpose: define a time-bounded, artifact-first loop that yields high-signal evidence fast.

Recommended loop (per role) — please confirm durations:
- Pre-screen (async, 20–30m reviewer time): blind-first artifact review against rubric anchors; preliminary score.
- Core interview (45m): structured, criterion-anchored Qs (2–3 criteria), live problem or case (thin slice).
- Work sample (60–120m async): tightly scoped task mirroring real work; candidates choose when to do it; clear rubric.
- Debrief (15–20m): multi-rater; record rationale + 1 disconfirming point per rater; preliminary verdict.
- Founder fit (30m): values/working-style alignment; risk/mitigation discussion on any critical gaps.

Notes:
- Total on-candidate live time target: ~75–105m; async work sample optional by role/stage.
- We can compress to a 1-day loop if urgent; or spread over 3–5 days to reduce candidate burden.
