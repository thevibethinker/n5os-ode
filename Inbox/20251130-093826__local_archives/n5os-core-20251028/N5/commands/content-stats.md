# content-stats

Show statistics for the content library.

## Usage

```bash
python3 N5/scripts/n5_content_library.py stats
```

## Output

- Total items count
- Breakdown by type (links vs snippets)
- Deprecated and expired counts
- Distribution by purpose and audience tags
