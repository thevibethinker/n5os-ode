## 🚀 Installation & Usage

### Quick Start

```bash
# Extract package
tar -xzf n5os-lite-v1.0-COMPLETE.tar.gz
cd n5os-lite

# Run bootstrap (interactive)
./bootstrap.sh

# Or non-interactive
./bootstrap.sh --non-interactive --name "Demo User" --workspace /home/workspace
```

### What Bootstrap Does

1. **Installation** - Copies all files to workspace
2. **Health Check** - Validates installation
3. **Onboarding** - Personalizes for user
4. **Validation** - Verifies setup complete
