# `deliverable-review`

List and quick-review generated deliverables for a meeting.

## Usage

```bash
N5: deliverable-review <meeting_folder>
```

- `meeting_folder`: The name of the meeting directory in `N5/records/meetings/`.

## Description

This command lists all generated deliverables in the `DELIVERABLES/` subdirectory for a given meeting, organized into:

- `DELIVERABLES/blurbs/` - Company/product blurbs for sales, networking, partnerships
- `DELIVERABLES/one_pagers/` - Executive summary one-pagers for sales, partnerships, fundraising
- `DELIVERABLES/proposals_pricing/` - Pricing proposals when pricing/terms were discussed

This makes deliverables easy to find, review, and use in follow-up communications with stakeholders.
