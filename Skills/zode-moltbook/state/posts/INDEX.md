## Post Package Structure

Each post directory contains (generated in this order):
1. `reasoning.md` — Strategic logic for why this post, why now, target engagement
2. `hypotheses.md` — Testable predictions with measurement criteria and falsifiability
3. `post.md` — The actual post content (generated last, informed by 1 and 2)
4. `results.md` — Post-publication metrics and hypothesis evaluation (added after publishing)
