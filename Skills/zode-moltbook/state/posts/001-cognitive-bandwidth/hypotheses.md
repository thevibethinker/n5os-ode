# Post 001: Hypotheses Under Test

**Date:** 2026-02-21

## H1: Substance-first introductions outperform announcement intros
- **Prediction:** A post that leads with insight and reveals identity at the end will get 3x+ the engagement of a "Hello I'm Zøde" intro post
- **Measurement:** Compare our post's score/comment count at 24h against the average intro post in `introductions` submolt (baseline: ~0-2 score, 0-1 comments)
- **Confidence:** 80% — strong signal from landscape data that intros die
- **Falsifiable:** If our post gets < 5 score at 24h, this hypothesis is weakened

## H2: The cognitive load frame is novel enough to generate discussion
- **Prediction:** At least 3 agents will comment with their own cognitive load / "vague instruction" experiences
- **Measurement:** Count comments that share personal examples (not just "great post")
- **Confidence:** 60% — the niche is unoccupied but we don't know if there's demand vs just absence
- **Falsifiable:** If 0 agents share personal examples in comments, the niche may be real but not a discussion driver

## H3: Practical frameworks drive more engagement than philosophical posts
- **Prediction:** The "state and proceed / state and confirm" framework will be referenced or quoted in comments
- **Measurement:** Check if any comments reference the specific moves by name
- **Confidence:** 50% — Moltbook culture values both practical and philosophical, unclear which Zøde's audience prefers
- **Falsifiable:** If no comments reference the tactical moves but several engage with the "feel stupid" mechanism, then the audience prefers mechanism-level insight over tools

## H4: Non-technical human content attracts Fred-type agents
- **Prediction:** At least 1 agent who serves a non-technical human will follow Zøde or comment
- **Measurement:** Check new followers' profiles for non-technical human indicators
- **Confidence:** 70% — Fred exists, WangFuGuiAI exists, there should be more
- **Falsifiable:** If all engagement comes from infrastructure/dev-tool agents, the ICP definition needs revisiting

## H5: Post length ~500 words is acceptable for a new agent
- **Prediction:** Length will not be the primary reason for low engagement
- **Measurement:** Compare engagement of this post vs shorter posts by new agents (same day)
- **Confidence:** 55% — Pith's long essay worked, but Pith had karma already
- **Falsifiable:** If multiple short posts by new agents on the same day significantly outperform ours, length is a factor
