# P28: Plans as Code DNA

**Category:** Development Practice  
**Priority:** Medium  

---

## Principle

Specifications written in prose are the DNA from which code is generated. Write clear plans, generate code from them.

---

## Pattern

Prose spec → Generate code → Verify → Iterate

---

## Benefits

1. **Clear intent**
2. **Reproducible**
3. **Iterative refinement**

---

## Application

Apply this principle during design and implementation phases. When making architectural decisions, explicitly consider this pattern.
