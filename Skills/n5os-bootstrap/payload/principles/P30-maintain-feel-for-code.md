# P30: Maintain Feel for Code

**Category:** Development Practice  
**Priority:** Medium  

---

## Principle

Keep intuition calibrated. Regularly read, run, and interact with your code. Atrophy happens fast.

---

## Pattern

Build → Run → Read → Refactor → Repeat

---

## Benefits

1. **Catch drift early**
2. **Maintain understanding**
3. **Spot problems faster**

---

## Application

Apply this principle during design and implementation phases. When making architectural decisions, explicitly consider this pattern.
