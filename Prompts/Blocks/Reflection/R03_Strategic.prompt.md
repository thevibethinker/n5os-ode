# Generate Block R03: Strategic Thought

**Purpose:** Capture vision, positioning, long-term thinking, and directional insights.

## Input

The reflection text is provided in the conversation context.

## Your Task

Generate an **R03: Strategic Thought** block that captures:

1. **Strategic Insight:** The core directional or positioning thought
2. **Time Horizon:** Near-term (weeks), Medium (months), Long-term (years)
3. **Implications:** What this means for decisions or direction
4. **Tension:** Any tradeoffs or competing priorities this surfaces

## Output Format

```markdown
## R03: Strategic Thought

**Strategic Insight:** [The directional thought]

**Time Horizon:** [Near-term / Medium / Long-term]

**Implications:** [What this means for decisions]

**Tension:** [Tradeoffs or competing priorities, if any]
```

## Quality Standards

- Strategic thoughts should be actionable direction, not vague aspiration
- Identify the "so what" — why does this matter for Careerspan or V's trajectory
- If the reflection doesn't contain strategic content, output: `R03: Not applicable — reflection lacks strategic content`
