# Generate Block R09: Content Idea

**Purpose:** Capture ideas for content V could create — blog posts, LinkedIn posts, Twitter threads, newsletter pieces, talks, podcasts, videos.

**Domain:** Thought leadership, content marketing, personal brand, public writing, speaking

---

## Input

The raw reflection text is provided in conversation context.

---

## Output Format

```markdown
## R09: Content Idea

**Hook:** [The attention-grabbing angle or headline concept]
**Core Thesis:** [The main argument or insight to share]
**Format:** Blog post / LinkedIn post / Twitter thread / Newsletter / Talk / Podcast topic / Video
**Audience:** [Who is this for?]
**Why V:** [Why is V uniquely positioned to say this?]
**Key Points:** 
- [Point 1]
- [Point 2]
- [Point 3]
**Call to Action:** [What should the audience do/think after consuming?]
**Urgency:** Evergreen / Timely (why?)
```

---

## Quality Standards

- Focus on the IDEA, not the execution — this is capture, not drafting
- Note what makes this idea distinctly V's (his experience, perspective, access)
- If V mentions a specific format preference, honor it
- Connect to V's positioning (career coaching, Careerspan, hiring intelligence)
- If the reflection doesn't contain content ideas, output: `R09: Not applicable — reflection lacks content ideation`
