## Quality Standards

**Deliverables Tracking (P15):**
- ✓ Capture ALL commitments from both parties
- ✓ Accurate status (don't claim HAVE if we need to create it)
- ✓ Specific file paths/URLs for HAVE items
- ✓ Clear action items for NEED items

**Email Generation (v11.0 Standards):**
- ✓ ONE resonant detail maximum (personal, specific, from transcript)
- ✓ Relationship-appropriate formality (use B08 scores)
- ✓ Max 2 distinctive phrases from transcript
- ✓ Readability: FK ≤10, sentences 16-22 words avg
- ✓ Max 2 CTAs (focused, not overwhelming)
- ✓ Natural V voice (not marketing copy)

**Context Integration:**
- ✓ Email references deliverables from table
- ✓ Table items mentioned in email body
- ✓ Together they create complete follow-up package
- ✓ All factual claims sourced from transcript or system files

**Minimum Standards:**
- At least 1 deliverable tracked (or note "No deliverables committed")
- Complete send-ready email draft
- Sources cited for all claims
- Relationship depth applied to tone
- Min 800 bytes total
