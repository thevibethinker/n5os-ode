# Quick Classify

Rapid document classification and filing.

**Process:**

1. Identify document to classify
2. Determine category:
   - **Knowledge** - Reusable insights, guides, references
   - **Lists** - Action items, todos, planning
   - **Records** - Meeting notes, correspondence, logs
   - **Documents** - Final outputs, deliverables
3. Apply appropriate emoji tags from `file N5/config/emoji-legend.json`
4. Move to target location with proper naming
5. Update relevant index files

**Quick Rules:**
- Records → staging, not SSOT
- Knowledge → permanent, reusable
- Lists → active work
- Documents → finished products
