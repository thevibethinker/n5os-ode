Careerspan Pilot — AI-driven JD → Candidate Triage

Objective

Run a short pilot to validate Careerspan’s integrated workflow for batch hiring: AI-generated JDs, role landing page to improve conversions, and candidate triage for first wave of applicants.

What we’ll deliver

- AI-synthesized JD(s) for target roles (data PM, data scientist) — informed by hiring manager notes
- Role landing page (interstitial) to improve magic-link conversion and present clear CTA
- Candidate triage report for first 20 applicants (match scores, top 5 recommended)
- Short demo session with HR + hiring manager to review findings and next steps

Timeline (3–4 weeks)

- Week 1: Collect hiring manager inputs + generate JDs and landing page draft
- Week 2–3: Post live listing, collect applicants, run initial triage
- Week 4: Deliver triage report + demo and decision meeting

Success metrics

- Time to first qualified candidate: target < 10 days
- Candidate match score: average > 0.7 (top 5)
- Conversion: view → apply conversion rate increase of 15% vs baseline

Dependencies

- HR approval for one live listing
- Access to hiring manager for a 10–15 minute intake
